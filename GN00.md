# 一

“熬制3小时45分钟后，加入蟾蜍腿五只、鼠尾草123克、红龙涎液22毫升……”

提耶利亚口中喃喃自语，手上麻利地按照配方把材料扔进面前冒着蓝色泡泡的大口径黑铁锅，锅底熊熊燃烧的木柴将她的脸庞映成了和屋外晚霞一样的颜色。

“继续熬制35分钟。”她伸手拨动猫头鹰闹钟定时器的眼睛，设置好时间。

拾起桌上羽毛笔，提耶利亚在羊皮纸笔记本对应的条目上打了个勾：“威力加强版沉睡毒药马上就要完成了。”

\
比起需要发挥想象力咒语作业，还有需要大量体力和注意力的扫帚飞行作业，药草作业当之无愧是提耶利亚的最爱。严格按照书本完成每一步要求是她最擅长的事情，通过各种反应公式和剂量计算炼出更大威力的药水以她的智商来说简直易如反掌。

今年是她在巫师研究所的202个年头，巫师袍领口的星星满了10颗。超过300岁的巫师学生在学院中已经很少见了，提耶利亚是其中一员。她倒不是不能毕业，是不想毕业，所以故意将“情商”这门课程大拖拖拉拉不肯修完。她想要和她指导员多相处一段日子。

没有理由继续拖下去了，大概明年就要被强制毕业了吧，真希望能和Veda一样成为这里的教员……

提耶利亚这样想着，念起静熄咒语，熄灭了铁锅下方的烧得正旺的柴火，又用冷却咒语将沸腾的一锅药水变成常温，装进玻璃瓶。

\
咚咚咚。

药草实验室的门响了几下，然后哗啦被猛地拉开。

“提耶利亚，你果然在这里。”

“老师！”

来者不是别人，正是提耶利亚的指导员Veda，整所巫师学校里她最尊敬、最喜爱的人。

\
Veda进门后，低沉着脸，用严肃的表情望着提耶利亚，然后一口气说道：

“提耶利亚，这件事很突然，你冷静一点听我说。本来你应该明年毕业的，可是天使组织今天下达了命令，让我们立刻马上抽调一名巫师上任。全学院资格最老的学生就是你了，所以校长决定让你提前毕业，成为天使。”

“啊？我？可是我、我还不够格呢。”

“嗯”，Veda扶了下鼻梁的眼镜，点点头：“你的确不够格，智商已经达到人类1000岁的水平，情商还停留在16岁。按照规矩是要等你的情商涨到18岁才毕业的，无奈学院真的找不出其他巫师，明天不提交候选人名字就要被那帮天使罚款……校长说最近全额缴纳学费的学生越来越少，贷款来上学的越来越多，学校的食堂都快揭不开锅了。我们也需要的你的工资，来偿还你欠学校的助学贷款。”

经过Veda提醒，提耶利亚才想起来自己也是欠这所学校钱的贷款人之一。 

“……但是老师，我的理想是留在这里，成为像您一样值得尊敬的魔法教师啊……”

“别开玩笑了提耶利亚！”

Veda打断提耶利亚，他的声音似乎带着一丝惊恐：“实不相瞒，我们学校里的老师快要比学生多了。教职员工里不会再有空缺容纳你了！……不过你也不要灰心。说不定、说不定哪天魔法会再次兴盛，学院涌进大批可以全额付学费的有钱学生。你是我教过最出色的学生，到时候我一定会向校长推荐你回来教书的。好啦，赶紧收拾东西，明天去上任。你赴任的地点，这个文件袋里有说明。这里面还有天使徽章、前往目的地的车票和船票。”

Veda拍了拍提耶利亚的肩，语重心长地说：“去那儿好好干，别给巫师学校丢脸。你的搭档很可能是个吸血鬼，咱可不能让他们给看扁了。”

啊，如果有什么比离开Veda更糟，那一定是离开Veda后还要和吸血鬼共事做搭档了。

\
我们所描述的这个世界，除了人类之外，还有很多奇奇怪怪的“超自然生物”。大部分超自然生物都随着自然进化被淘汰掉了，活该，谁让他们想要超越自然呢。只有四种种族通过了自然的筛选存活至今，并和世界的霸主人类形成了“友好”共存的局面。

他们是吸血鬼、巫师、狼人和恶魔。

人类给这些超自然生物提供“崇拜”、“敬仰”与“畏惧”，超自然生物们则从人类的感情中获取了与自然进化抗衡的能力。

在远古时代，四个种族为了守护他们生命的来源，避免人类作死灭绝，共同创建了一个叫“天使”的组织。他们向人类世界派遣名为“天使”的员工，保护一方水土的平安，类似于中国古代传说中的“土地公”和“土地婆”。他们之所以敢使用“天使”这个烂大街的名字，纯粹是因为这个世界根本没有上帝。后来，因为某些无法考究的原因，狼人和恶魔宣布退出，该组织目前仅剩下吸血鬼和巫师两个派别。

吸血鬼和巫师在所拥有的异种族能力方面可说是两个极端。吸血鬼身具无与伦比的“力量”，能够控制包括重力在内的一切受牛顿三大定律影响的力量。嗯，幸好他们还不能控制热力学和量子力学。而巫师则拥有操纵与变换万物的“魔法”。没错，万物，除了吸血鬼。

“那群该死的吸血鬼，不仅拥有了至高无上的力量，还拥有全世界最高的魔抗值。”

——提耶利亚至今都记得她的咒语老师在课堂上用粉笔头用力敲着黑板、说出这句话时脸上那愤恨又无奈的表情。

\
目前“天使”组织元老会的九大天使里，吸血鬼占了5席，巫师占了4席，看起来是“力量”占了上风。但因为首座“炽天使”是个巫师，其实两个派别仍是势均力敌。 比起三足鼎立互相牵制，“两党相争”无论在哪个世界都是最容易撕破脸的局面。“天使”组织也不能幸免。吸血鬼和巫师两派较了几千年的劲，互相看不顺眼。他们都认为自己才是“天使”的老大，是应该统领对方的人。

\
Veda把上任需要的文件交到提耶利亚手里，离开药草实验室前又转过身叮嘱他的学生：“还有啊，和搭档好好相处。我们的魔法在吸血鬼身上的效果极其混乱，有些法术还会被他们反弹回来，还是尽量避免和他们起冲突比较好。”

# 二

经过海上三天的颠簸旅程，提耶利亚终于到达了她成为“天使”后的第一个工作地——位于托勒密海北部的GN00小镇。

\
船舶停港的时间点正值晌午。在GN00小镇东南面建有一个小型港口，供来往客船停靠，另外还有渡轮往返于周边岛屿。这港口大部分时间用于运输游客，很少有大宗商品在此处卸货交易，无法比拟深水港的热闹程度。放眼望去，用石头搭建的港口沿岸只有稀稀拉拉几个拉行李的挑夫。与地面的冷清截然相反，此刻天空上正悬挂着毫无遮拦的大太阳，猛烈地炙烤着灰色的海水和岸堤，四周的空气仿佛是从加热腌大马哈鱼的蒸锅里冒出来的一样。

提耶利亚走出船舱，头戴尖尖的巫师帽，身披宽大的黑色巫师袍，两件行头便足以标示她作为巫师这一超自然生物的身份。对于眼前这片距离巫师岛2000海里的新大陆，提耶利亚完全提不起好奇心。一方面因为她对人类本来就没有兴趣，另一方面则是身体原因。昨晚海上突起的大风浪搞得她整宿没合眼，一直坐在船长室诵念风平浪静咒语、操纵法术风与之抗衡。保护所在航船安全是做巫师的基本责任，倘若某艘载有巫师的船不幸翻了，那将成为整个巫师族的耻辱，顺便成为外族消遣他们的笑话。

虽然提耶利亚这么做八成是为了自己的小命，两成是为了巫族荣誉，和整船几百号的人类关系不大，下船时船长还是对她千恩万谢，不仅将船票钱全额退还，还额外送了两张烤饼作为礼物。烤饼的色泽很漂亮，味道闻起来也不坏，提耶利亚没吃早饭，按理说应该饿了，可是裹着厚重的巫师袍浸泡在发酵烤箱般的日光浴中，她实在提不起胃口。不过，她仍旧接受了这位白胡子老人的好意，将烤饼放进身后背包，拎起盛着衣物和咒语书的皮箱，提耶利亚走下了扶梯。

\
走出港口用鲸鱼骨制成的大门，提耶利亚掏出塞在巫师衬里中的地图。GN00小镇的“天使”派出所离港口有10公里，要是平时这点距离对提耶利亚并不能造成障碍。可是现在她拖着个沉重的大皮箱，倘若强行用扫把飞行，扫把搞不好会罢工断掉顺带摔断提耶利亚的三根肋骨。

“唉，算了，走着去吧。”

提耶利亚把心一横，刚准备拖拽起比新鲜死尸还重的皮箱向目的地迈进，就听见一个女子高亮的呼喊：“厄德桑！厄德桑！”

\
提耶利亚扶了扶眼镜，逆着声音传来的方向看过去，只见一辆马车疾速驶来，从车窗探出一个脑袋和一只向上挥舞的手。

马车夫在她身边勒住马。车门砰地撞开，一个褐发少女从车厢里跳了下来。

“您这身打扮一定是厄德桑了是吧！我是米蕾娜·瓦斯提，副镇长的女儿，我父亲派我来接你。快上车吧，我送您去天使派出所。”

米蕾娜有一双一双扑闪扑闪的大眼睛，提耶利亚敢打赌她那对眼睛占了有整张脸的三分之一的面积。

提耶利亚张口欲回答，可是一个字都没来得及蹦出，就被米蕾娜拉住手拽上了车按到最里面的座位坐定。

车夫挥鞭启程。

“我的箱子！”提耶利亚转头，用近乎哀嚎的语气冲着后窗喊道。

“哎呀呀，对不起！拉塞，你去抬一下箱子。”

前排那个叫做拉塞的车夫听见后，一言不发再次勒住了马，走下车，像抗一袋棉花一样扛起提耶利亚的沉重皮箱放到车厢后的行李架上。提耶利亚这时才发现拉塞胳膊上的肌肉厚度有多么惊人。

\
路上，作为市政厅秘书的米蕾娜给提耶利亚讲述了GN00小镇的基本情况。原来，GN00本是个相当和平安宁的地方，并不需要天使坐镇，但从5年前开始，不知为何，这里隔一段时间总有人离奇失踪，天使组织才开始派人来此地驻扎。

“您之前的那两位天使大人都是巫师，一位得病死了，另一位是自己去海里游泳，淹死了。唉，他们都是好人呐。愿他们安息。”

米蕾娜在胸前画个十字，叹了口气接着说：“不过旧的不去新的不来嘛。能有您这样漂亮的天使驻扎在此也让我们镇蓬荜生辉。哦对了，您的搭档昨天已经到了，也是一位非常漂亮的人物呢。”

听到新搭档，提耶利亚昏沉的脑袋才有了精神：“哦，她也是位女性吗？”

“不，是男性，是位非常帅气的吸血鬼大人。他还吻了我的手背呢！”

\
会装绅士的吸血鬼，不能再差了。

上任路上的闲暇时间，提耶利亚一直在脑子里琢磨如何能压制住她的搭档占到上风。要知道每一组巫师和吸血鬼的搭档都是两族争斗暗涌的缩影，如果自己输掉，很可能给全巫族今后的命运带来巨大的影响。

争取人类的爱戴，提高巫族声誉，从而让巫族获得更多人类的“精神力量”，是提耶利亚给自己定下的“小目标”。

要战胜吸血鬼这种既有钱，又会耍帅，还爱装绅士的生物，的确有些困难。好在提耶利亚生平最爱向困难发起挑战。

\
将提耶利亚送至天使派出所的门口后，米蕾娜递给她一封请柬。

“是镇长修亨伯格大人要我转交您的。明天他家里举行宴会，请您务必到场。”

\
与米蕾娜和拉塞告别后，提耶利亚推开了派出所的木门，迎面扑来一股凉意，和户外的空气极其不相称。

派出所是两层建筑，底层是办公区。房间不大，侧面是一个摆满文件夹的铁架子，正中间是两张桌子。明明是中午，正对门的窗户却拉上了窗帘，屋内光线很暗。

提耶利亚径直走了进去，把箱子放在一旁，从袍子里抽出法杖唰地拉开窗帘。

“怎么，吸血鬼到现在都见不得阳光吗？我怎么听说你们已经将这个毛病都进化好了？”

她扭过头，用讥讽的语气对那个坐在左边桌子旁的“新搭档”说道。

她的搭档把脚从桌子上挪到地上，身子动了一下，阳光照到了他的脸上。

米蕾娜说得没错，这家伙长了一张颇受女人欢迎的脸。提耶利亚心里暗自骂道。

“我并不怕阳光，只是在睡午觉而已。”他笑了一下，露出两侧的尖牙。“你就是提耶利亚·厄德吗？我叫洛克昂，洛克昂·史特拉托斯，请多指教咯。”

洛克昂伸出手，他手上是和提耶利亚一样的纯白手套。

两人隔着手套轻轻握了一下手，提耶利亚心不在焉地想着这笑容能俘获全镇上至八十下至八岁的女人，她已经丧失了一半的“选票”。

“好啦，招呼打完了，麻烦你把窗帘再帮我拉上，我继续午睡。”

洛克昂说完，扯了扯黑色礼服背后的斗篷，打了个哈欠。

“喂！现在是上班时间吧，要睡觉等下班再睡，天使组织给你发工资不是让你在办公室睡觉的。”

“这话从一个上任迟到的人嘴里说出来，没什么说服力啊。”洛克昂用慵懒的声音回答。

“我、我预定的上任时间是明天！”

“我也一样。”洛克昂嘴角向上牵动，又露出了两颗尖牙。“按规定，现在是我的休息时间，既然不能在办公室睡，那我上楼睡好了。”

“等等！”

洛克昂刚准备起身，听到提耶利亚的呼斥，又坐回了椅子。

“现在我们两个人都在这里，趁这个机会把分工明确一下吧。每队搭档都需要选出‘队长’，既然我们都是第一次担任天使，那么队长就轮流做吧……”

“我不是第一次哦”，洛克昂轻描淡写地打断提耶利亚，“我之前曾做过83年的天使。经验虽然也不多，但我可不是你这样的新手。而且……”

洛克昂掏出了他的上任聘书，将有字的一面对着提耶利亚：“我的聘书上明确写着‘担任天使队长’哦。”

提耶利亚气结。她没想到天使组织已经指定让对方担任队长一职。

不过嘴上仍不肯认输：“哼，我会在今后工作中考察你的能力，评定你是不是适合担任队长。”

“能力？你在质疑吸血鬼的能力？”洛克昂哈哈一笑，“那我让你见识一下好了。”

说罢，他抬起右手，食指勾勒出一个很小的幅度。

瞬间，提耶利亚感觉背后一股强大的力量猛地推动她的身体，不由自主向前平滑移动，像是在高山雪道失速般无法自控的恐惧感涌上心头。

\
无论是洛克昂还是提耶利亚，都没发现派出所办公桌前有一块地板因为海边经年的潮湿，一头翘了起来。

这块翘起的地板把提耶利亚绊了一下，而这一绊并不在洛克昂的计算中。

提耶利亚整个人趴在洛克昂身上，自己的巫师帽飞了出去，还差点把对方连椅子一起掀翻在地。不过洛克昂还是控制住了椅子的势能，只让它后滑了几厘米，没有让两人出丑。

他伸手拦住已经坐到自己腿上的提耶利亚的腰，呼出一口气。掌控了眼前状况后，他贴到提耶利亚耳边笑着低声说：“没想到你还挺热情。”

不过下一秒他就笑不出来了。他吸进的空气中混杂有提耶利亚被太阳烘烤后流出的汗液味道，洛克昂脑子里发出嗡地一声巨响，本能地把提耶利亚推到了地上。

“你、你做什么啊！”

提耶利亚坐在地上，已经出离愤怒了。活了300多年，她第一次遭受这种侮辱。

“啊，对不起。”

洛克昂站起身，撇下一句简单的道歉，直接往楼上走。

\
提耶利亚从地上爬起来跟了上去，一把抓住洛克昂的黑色斗篷：“你别跑，说清楚再走！”

“刚才是个意外，我只是想移动一下你的位置，没想到你会摔倒。”洛克昂头也不回地说，语气听上去很着急。“不过你最好以后每天洗澡。”

“啊！你什么意思！”提耶利亚的气愤丝毫未减。

“你汗液的味道对我有刺激作用，会将我唤醒。”

“唤醒？什么唤醒？用我能听懂的语言说话行吗！”

洛克昂终于转过头，凝视着提耶利亚，一字一顿说：“唤醒，就是性意味上的觉醒。我闻到你的气味后，不仅会想喝你的血，还会想要上了你。”

# 三

提耶利亚把箱子抬进派出所楼上插着自己名牌的房间，一头瘫倒在床上。一想起刚才洛克昂说“要上了你”时闪着幽暗绿光、好似喷出火焰的眼睛，她又从床上跳起来跑到门边，再次确认房门已经锁好，门上的链条也100%套入了门闩。

“幸好房里有独立的洗澡间……明早之前都不想出房门了，晚饭就吃那两块烤饼吧。”

傍晚，提耶利亚在独立浴室里待了很久，耗费了约一吨的清水和一整块肥皂，恨不得揭掉自己一层皮。夜里她屡次惊醒，恍惚中依稀觉得吸血鬼站在床前邪魅地笑着，尖牙像明晃晃的匕首，下一秒就会咬上自己的脖子。

\
次日清晨，提耶利亚瞪着没怎么阖过的双眼一巴掌拍停了在脑袋上方飞来飞去的猫头鹰闹钟。

洗漱时，她望着镜子里的黑眼圈喃喃自语道：“天使最短的任期是五十年，五十年之后就可以申请调职。没关系的，只有五十年而已嘛，一眨眼就过去了。”

说罢，她用力眨了眨眼，作为对自己的鼓励。

\
上午9时，提耶利亚准点坐在办公桌前。

“队长”洛克昂迟到了3分钟，他迎着提耶利亚鄙夷的目光在椅子上坐好，打了个哈欠。

\
“你迟到了。”

“啊，是吗”，洛克昂看了看文件柜旁边的座钟，“差不多啦，不要总在意这些小事。‘天使’是不需要守时的。”

“荒谬！……”

“好啦好啦，以后我会注意啦。”洛克昂像知道提耶利亚将要开始长篇大论的教训一般，果断地打断了她的话。

“话说，我还从来没和巫师共事过，之前的搭档也是个吸血鬼。你们巫师都会些什么啊？”

“巫师的本领在超自然生物界是常识吧。难道你在学校没有学过吗？”

“学校？”洛克昂耸耸肩，“吸血鬼没有学校，我们的能力都是天生的。力量、飞行、狩猎和听觉，从我成为吸血鬼的那一刻就存在了。唯一需要学习的就是如何在人群中控制想要喝鲜血的欲望，这对我来说并不困难，我的嗜血程度在同类里算是比较弱的。”

看见提耶利亚露出鄙夷加怀疑的表情，洛克昂用抱歉的态度解释道：“昨天的事情只是个意外。你看，我对你清洗干净的身体提不起任何性致，我不会对你出手的……是真的，别用那种眼神看我啦。所以你们巫师是有学校的咯？在学校里都学些什么？”

提耶利亚轻哼一声，将目光收回：“我在巫师学校中，最擅长的是药草学。”

“药草学？是去山上采药吗？”

“当然不是啊！”提耶利亚表情变为无奈，扶着额大大叹了口气：“药草学是从上古炼金术演变过来的，是涵盖了医术、蛊术、冶炼、物质变换的一门巫术。”

“噢噢，听起来很厉害。这么说你会治病咯？”

“普通的疾病和解蛊是没有问题的，但也不是包治百病。”

“还有呢？你们会占卜吗？”

“为人类占卜是被巫师协会明确禁止的！只有那些乡野巫师才会做这种事。”说到乡野巫师四个字时，提耶利亚的语气是极尽轻蔑的。“我们还会变形，还有在一定程度上操控天气，例如风向和雨水，以及使用一些咒语……”说到这里，提耶利亚闭上嘴。在吸血鬼面前，巫师的咒语可能失去效果或产生混乱，这不是一件值得夸耀的能力。

“欸，这样啊，你懂的还真多。”洛克昂点头称赞。

提耶利亚挑起眉毛笑了一下，表示对洛克昂赞赏的认同。

“好啦，你的情况我大致了解了。”洛克昂站起身，整理了一下环在颈部的白色风琴褶领巾，“那我们出去巡逻吧。”

“巡逻？为什么？”

“你的问题还真多。我们’天使’的工作内容其实和人类的‘警察’差不多啦，定期出街巡逻，盘查可疑人物，不过这个小镇有些特殊……”

“特殊？什么特殊？”因为洛克昂总不把话一次说完，提耶利亚已经有些不耐烦了。

“我有种感觉，”洛克昂压低了声音，“在我们之前的那两个’天使’，死因蹊跷，必有隐情。”

\
提耶利亚锁上派出所大门，小跑跟上已经走了50米远的洛克昂，拉住他的衣袖问道：

“到底哪里蹊跷？有什么隐情？”

“我有一些发现……时机到了自然会对你说的，现在还不是时候。”

又在故弄玄虚。提耶利亚停下脚步，微微嘟起嘴。洛克昂没有理她，大踏步继续着“巡逻”。

看着洛克昂的背影，一丝不甘心的情绪从提耶利亚心底油然升起。对方是礼服加披风的潇洒装扮，自己则是滑稽的尖帽子加上宽大及地的黑斗篷。巫师协会设计制服的人是猪脑子吗，这样的穿搭根本无法吸引人类的仰慕之情嘛！呃，手上还握着一条飞行扫把！提耶利亚拿出魔杖，气呼呼地将扫把变小，塞进口袋里。

\
你看看，这一会儿工夫，已经有7个小姑娘对洛克昂行注目礼了。

洛克昂对她们一一脱帽致意。

恶心。这么爱现，不如去做吟游诗人啊。

提耶利亚心底暗暗骂着，嘴里不经意发出“啧”的一声。

她忘记再细小的声音都逃不过吸血鬼的耳朵。

洛克昂转过头：“有什么事吗？”

“没、没什么。”提耶利亚向队长挤出一个假笑。“那个、我们要巡逻到哪里啊？”

洛克昂抬起戴着白色手套的右手，指着远处的青翠山峰说：

“把镇子转完。一直走到那边山脚下，就可以用飞行术回派出所了。”


\
GN00小镇东南部是港口，其余三面均被山包围。山脉如同海星的爪子一样延伸到被海水冲刷形成的平原上。虽然山不算高，路也好走，却也难免给小镇与外界的交流带来些许不便。因为半封闭的地理位置，小镇维持着自给自足的状态。尽管人口不多，教师、医生、护卫兵队仍全部由本地人担任。不仅山间和平地的瓜果物产丰富，海里还有取之不尽的鲜美鱼类，除了供镇上的人食用，也有不少行脚商人闻名前来采购当地特产再运输到外地，总体说来是个相当富饶的小镇。若不是几年来频繁发生的人口失踪事件，这里还真可以称得上世外桃源。


\
提耶利亚走在小镇中心宽阔的马路两边供行人行走的石板路上，周围是用红瓦或青砖建造的低矮房子。居民们目送二位天使巡逻经过时，都会露出亲切的笑容表示敬意。

太阳已经升得很高。这个海边小镇，早晚在潮水和海风的笼罩下凉爽宜人，日上三竿便会形成蒸笼般的气候。

提耶利亚掏出了魔杖，小声嘀咕着咒语。

“你在叽里咕噜说些什么呢？”

提耶利亚没好气地白了洛克昂一眼，答道：“冷却咒语！我可不想出汗。”

“你觉得热？”洛克昂一副不可思议的表情。

“是啊。难道你不热？”

“吸血鬼是凉血生物。除非用肌肤触碰，否则我们不会感知到热度，所以对大气的温度没什么概念。”洛克昂摘下白手套，“不信你摸一下。”

提耶利亚看着洛克昂如白骨雕成修长的手，忍不住好奇心，也脱下手套，伸出指尖碰了一下。他的肌肤触感如同玉石，光滑且清凉。

“好凉！”提耶利亚惊叹。

“你的指尖也好热。”洛克昂戴好手套，笑着回应说。


\
途径一家面包作坊时，洛克昂停下来，掏出银币买了一个刚出炉的吐司，并告诉店主不用找零。他把装着吐司的纸袋抛给提耶利亚：“吃吧，我听见你肚子在抗议了。”

提耶利亚伸手摸了摸腹部，是有些饿了。什么声音都逃不过洛克昂的耳朵。


\
“你不吃吗？”

“吸血鬼不需要进食。”洛克昂继续往前走，头也不回地说。

“哼，既不会出汗，又不用进食，你们吸血鬼的生活还真方便。”

提耶利亚咽下一口吐司，等待洛克昂的回话。

她什么都没等到。洛克昂默默凝视着海面上飘荡的渔船和天空中翱翔的海鸟，不再作声。

# 四


傍晚，洛克昂和提耶利亚准时来到镇长伊奥利亚·修亨伯格位于小镇北部的宅邸，应邀参加晚宴。

\
修亨伯格的大宅位于北部山丘的半山腰上，站在这里可以远眺整个小镇乃至港口的海岸线。房屋有三层，灰白色的外墙上镶嵌着黑色边框的大窗户，红色琉璃瓦尖顶，比小镇上的普通民宅的确要豪华一些，可比起大城镇中富贵人家的豪宅还是相去甚远。不过，房屋外由木栅栏圈起的一片花园，其面积颇大，乃是繁华都市中那些豪华大宅无法比拟的。


\
出门迎接两位天使的人是提耶利亚昨日已经见过的车夫兼管家，拉塞·艾恩。

“老爷一直在等候二位，请跟我来。”

他只说了一句话，缺乏招呼贵客应有的热情问候与寒暄。这男人说话的字数和他身上的肌肉群呈反比，提耶利亚想。

花园里错落有致地种植着矮苹果树、紫丁香和玫瑰花丛，被精细修剪过造型昭示着它们是由专人悉心照料的家养植株，而不是山野草林里的野生灌木丛。沿着细长的石子路穿过花园，带有圆形把手的黑漆铜质大门正大大敞开着，对洛克昂与提耶利亚的到来表示欢迎。


\
提耶利亚以为镇长今晚宴请的客人只有新上任的两位天使，直到看见挤在会客厅里的十几个脑袋后，她才知道自己错了。

这间会客厅的面积不算小，装潢与房屋外墙一样朴素无华，家具全都是极其常见的样式。橡木圆桌旁围坐着五位客人，另外还有三人站在窗前，而此次宴会的主人——伊奥利亚·修亨伯格，正端坐在壁炉前的黑色羊皮沙发上，与一个绿色头发的俊美少年交谈着。

见到洛克昂和提耶利亚进门后，美少年立刻搀扶着腿脚不便、需要拄拐的光头镇长站了起来。


\
“欢迎二位光临寒舍。”

修亨伯格和他挑选的管家一样，是个不喜欢说废话的人。头顶的毛发比他的话语还要稀少，棕褐色的胡子却依旧茂盛。从他脸上的皱纹推测，年纪少说也有六十岁了，精神矍铄，双眼如同利剑一般有神。

“诸位，这两位就是被派到我们镇上的新任天使，吸血鬼洛克昂大人，和女巫提耶利亚大人。”

镇长低沉的嗓音让整个屋子安静下来，宾客们从桌前和窗边聚集到他身边，围成一圈。

按着他本人的行事风格，修亨伯格用简洁高效的语言向两位天使一一介绍了所有宾客的姓名和身份。提耶利亚一边暗记他们的姓名，一边仔细观察。

和他交谈的绿发美少年名叫利冯兹·阿尔马克，是他的养子。副镇长伊恩·瓦斯提和其夫人、镇小学校长琳达·瓦斯提，以及他们的女儿、市政厅秘书米蕾娜·瓦斯提。扎着长长马尾辫带着眼镜的斯文男人名叫比利·片桐，是镇中学教师。小镇唯一的医生亚瑟·古德曼是个黄毛胖子，脸上一直堆砌着热情的笑容，可是搭配上他松弛的下巴、布满毛孔的鼻头和下垂的眼袋，那笑容总让人觉得不怀好意。还有佣兵队长阿里·阿尔·萨谢斯，长长的红色卷发乱糟糟的，好像乱窜的火焰，面容精悍，身上凸起的肌肉被红色皮衣紧紧包裹，其他宾客被介绍到时都会上前与两位天使握手致意，他却一直站在窗前未曾挪动，仅以目光斜睨着洛克昂提耶利亚。提耶利亚觉得他比洛克昂更像“鬼”。不，还是“魔鬼”这个词更贴切一些。

除了这些“公职”人员外，修亨伯格还邀请了两位隶属民间的客人。一位是碰巧来到此地而受邀参加宴会的冒险者笛卡鲁特·谢尔曼。他身材高挑、白发红眼，带着一脸的自信和桀骜不驯。另一位则是小镇最大的富农家族继承人，亚历杭德罗·科纳。在修亨伯格介绍他的时候，提耶利亚忍不住多打量了几眼。此人无论身材还是脸庞都堪称完美，举止谈吐优雅有礼，浑身上下无一处不散发着高贵的气息。传说中的“高富帅”就是这个样子吧。而且他年方三十尚未娶妻，肯定是镇上年轻女子竞相追逐的对象。这下洛克昂有对手了。想到这里，提耶利亚不禁在心底笑出声来。

修亨伯格介绍完所有宾客后，拉塞进入会客室。他向修亨伯格鞠了一躬：“老爷，晚宴准备好了，请各位去餐厅用餐。”

\
所有的宾客在长条宴会桌上分坐成两排。修亨伯格的位置在长桌最中间，他的左手边是“天使队长”洛克昂，右手边是远方来客笛卡鲁特。拉塞行走在席间，负责斟酒和加菜。宴席的主菜有三种，分别是羊肉、猪肉和深海鳗鱼，主食是烤饼，饮料为红葡萄酒，佐餐水果有石榴和苹果，从内容的丰富程度上无可挑剔，味道也相当不错。主人还贴心为洛克昂准备了一杯新鲜的羊血。

“是今天下午刚刚宰杀的羊，羊肉在诸位的盘内。我让拉塞在血液中加了水蛭防止凝固，希望能合吸血鬼大人的胃口。”

“欸……”

洛克昂举起装着羊血的玻璃杯，借着头顶上方大型悬吊烛台成排蜡烛发出的光亮，晃了晃杯中血液。

“嗯，好血。镇长有心了。”


提耶利亚斜瞅着洛克昂，用眼神说你不是真的要在人类面前饮血吧。洛克昂也回盯着提耶利亚，用眼神答复我可不能辜负人类的一番好意啊。他一口气饮干了杯中血，然后用餐巾擦了擦嘴角。

提耶利亚看着身旁餐巾上残留的血渍，皱起眉，屁股往另一边挪了挪。


\
宴会很热闹，或者说是“吵闹”也不为过。发言最积极的是笛卡鲁特，他滔滔不绝地讲述着自己十几年环游世界、在各大洲遇到的奇闻逸事与风土人情。提耶利亚对这些故事兴趣不大，只顾着埋头吃菜，直觉告诉她这人在夸大其词。米蕾娜和比利倒是对他的故事非常感兴趣。尤其是比利，他请求笛卡鲁特在小镇上多停留几日，和他一起将这些故事编纂成教材供学生阅读。

洛克昂向镇长讲述了他上一次担任天使工作的基本情况。从他们的对话中，提耶利亚了解到他是从战争频发的南大洋洲调任过来的。

镇医生亚瑟·古德曼坐在提耶利亚对面，时不时与提耶利亚攀谈一两句。除了人类浅薄的医学，他对巫医术似乎也有一点研究。此人开口邀请提耶利亚在方便的时候去他的医馆莅临指教。“啊，一定”，提耶利亚漫不经心地敷衍着。古德曼笑得很开心，大概他没听出提耶利亚是在客套。

修亨伯格听见了医生与提耶利亚的的对话。他向提耶利亚提起，在他的藏书室里也有几本有关巫医学的藏书。

“是上一任天使赠与我的，他对我说也许什么时候有人能用得上。”

询问了书名后，提耶利亚诧异地发现自己竟有一本尚未读过。

“不介意的话，女巫大人可以在宴会结束后去书房阅读，借走也是可以的。”

提耶利亚微笑着向镇长表示了感激之情。

这句“谢谢”是她对伊奥利亚·修亨伯格说的最后一句话。

\
读者对于此事大约并不会感到意外，但故事里的这些人物还是受到了不小的惊吓：镇长伊奥利亚·修亨伯格被杀死了，死在了自己的卧室里。

\
我们还是按顺序讲述吧。事情的经过是这样的：

晚宴结束后，副镇长瓦斯提一家先行告辞。提耶利亚在拉塞的带领下进入位于二楼的藏书室。洛克昂则不知去向。

提耶利亚完全沉浸在书本里，她翻了大半本书后，听见书房的钟声敲了九下，才醒觉时间已经不早。

拿起书本，提耶利亚离开了藏书室，想着下楼后应该能碰到主人，要向他打个招呼再次道谢。

她一径走到大门口，都没发现一个人影。


\
“奇怪，人都哪去了？难道洛克昂也走了吗？那家伙居然不等我……”提耶利亚口中小声嘟哝着。接着她听见门外花园传来了洛克昂的声音：

“冤枉。我可是一直在这里等你呢。”

\
声音是从门外的花园传来的。提耶利亚走过去，见洛克昂长身直立站在花丛中。她问：“你在这里做什么呢？”

“这些玫瑰开得正漂亮，没人欣赏不是太可惜了？我在陪它们说话。”

提耶利亚嗤笑一声：“啊？陪玫瑰说话？你脑子没坏吧。”

她的问题是发自真心的。刚刚才大口喝下羊血，现在又对着几朵无人欣赏的玫瑰怜香惜玉，这绝对不正常。不过，她的念头很快转了过来：喂，他可是个吸血鬼啊，用“正常”的逻辑来要求他，似乎有些过分了。

提耶利亚扬了扬手中的书：

“书我准备借走。时间不早了，我们该回去了。我刚才下来的时候没看见镇长，果然还是应该和主人打声招呼再离开吧……”

“那我们进去找找他。”

\
于是二人返身回屋。可是还未迈进大门，就听见从楼上传来的杂乱脚步声。今晚宴会的客人：利冯兹、亚历杭德罗、萨谢斯、笛卡鲁特、比利，他们成群结队冲下楼来，脸上神色相当之古怪。

利冯兹一见到提耶利亚和洛克昂，俊美的脸庞瞬间扭曲。他颤抖着伸出手，指着吸血鬼和女巫，用悲痛欲绝的腔调大声喊道：“就是你们！是你们杀了我的父亲！”

# 五

利冯兹一行人将洛克昂和提耶利亚团团包围，不知发生了何事的两个天使面面相觑。

洛克昂先开口：“你说什么？杀了你的父亲？你的父亲是……”

提耶利亚接过话：“难道、难道是镇长？”

洛克昂说：“啊！镇长死了？！”

提耶利亚说：“怎么会！刚才不是还好好的吗！”

洛克昂说：“难道他有心脏病？”

提耶利亚说：“不像啊，我观察过他的面相，他各个内脏都很健康……”

\
“你们够了！”利冯兹愤怒地叫嚷：“别在这里说相声了！我父亲是被杀死的！”

两个天使异口同声说：“什么！被杀死了？！”

对于只见过一面、交谈过一次的镇长的死亡，提耶利亚感到万分诧异，不过绝对够不上“伤感”的程度。她歪着头问洛克昂：“是你干的吗？”

“不是。”洛克昂平静地否认，“是你吗？”

提耶利亚摇头：“不是我。”

“很明显，不是我们干的，凶手另有其人。”洛克昂直视利冯兹，十分肯定地说。

\
“啊？就凭你们一句话？”利冯兹露出难以置信的表情，难以置信到提耶利亚甚至在他原本应该悲痛的脸上看到了一丝嘲笑。“我们凭什么相信你们！”

\
“除非自身生命受到威胁，天使是严禁杀人的，就算是罪大恶极的人也不可以。否则，这枚天使徽章会变成红色，我们会受到’天罚’。”洛克昂从胸前口袋掏出纯白的、有两只翅膀水印的天使徽章举到利冯兹眼前。

他用胳膊肘捅了捅发呆的提耶利亚：“快把你的徽章掏出来给他们看。”

提耶利亚闻言，“哦”了一声，从巫师袍里取出同样一枚纯白的天使徽章。

“你们要是不信，可以写信给天使组织询问，或者请他们来调查都可以。但是我认为当务之急是找出杀害镇长的真凶吧。”

比利点点头：“我觉得洛克昂大人说的有道理。他们今天刚刚上任，和镇长大人无冤无仇，没有杀人动机。”

整晚没说过几句话的萨谢斯忽然开口：“杀人动机？哼，我以前在战场的时候，见过很多士兵仅仅为了乐趣而杀人，杀人这件事根本不需要理由。”

比利反驳道：“战场是一种特殊的环境，人长期浸淫在里面会丧失理智。这里又不是战场，两位天使大人也不是士兵……”

利冯兹打断了比利：“是啊，他们连人都不是，他们是超自然生物，人类哪里猜得透他们的想法。”

\
洛克昂轻咳一声，镇定地说：“这么看来，你们没有任何我们是凶手的证据。仅凭‘我们是超自然生物’这个事实，就得出推断说是我们杀了镇长，也未免太荒谬了。天使的职责是守护小镇安全，这起发生在镇上的杀人事件，我们有权介入调查。”

\
笛卡鲁特的手指从他一头柔顺的白发中穿过，又回到腰间。

为了给屋内剑拔弩张的气氛解围，他缓缓开口说道：“虽然我是个外人，无权过问镇上的事务，但是根据我周游列国的经验，那些有天使驻扎的城市啊，天使们的权利都是很大的。虽然失去伟大的镇长，无论对阿尔马克少爷，还是对这个美丽的小镇都是很大的打击，但为了不让镇长死不瞑目，更为了小镇的和平安宁，让天使尽快介入调查，找出真凶才是正确的选择啊。”

\
利冯兹低头沉默了几秒钟，仿佛在思考笛卡鲁特说的有没有道理。然后他抬起头，甩了一下镶着精致花边的喇叭袖口，牙齿缝里狠狠挤出一句话：“如果找不到凶手，我会向天使组织上报你们的失职，到时他们也会彻查你们到底有没有使用超自然能力杀人。”

\
来自利冯兹的阻碍消失了，洛克昂和提耶利亚上楼，到达案发现场。修亨伯格的房间位于三楼的东边，紧邻楼梯的位置。管家拉塞和医生亚瑟·古德曼在发现尸体后一直守在这里，他们宣称没有人动过尸体或现场的任何东西。

一进屋，浓烈的血腥气味让洛克昂的鼻翼不由自主翕动了几下。提耶利亚发现了搭档的异常，主动走近尸体进行查验。

伊奥利亚生前是威仪庄严的镇长。他的肉身在其死后也以同样端庄的坐姿陈列在书桌旁宽大的椅子上。致命伤显而易见。他光亮的脑门上有一道很深的伤口，颅骨凹陷，血迹一直延伸到地上。尸首旁边的地上随意丢弃着一把武士刀，刀锋沾染着血迹，无疑便是凶器。

“一刀致命，没有挣扎的痕迹。有两种可能，要么凶手是镇长很熟悉的人，要么被杀的时候他在睡觉。”

提耶利亚检查完伤口和尸身，如此总结道。

洛克昂环视房间的环境。“房间很整齐，没有被翻动过的迹象。所以凶手的杀人动机，比起盗窃，私人仇恨的可能性更大一些。”


\
提耶利亚从尸体挪开视线，问拉塞说：“镇长最近与什么人结过怨吗？”

拉塞摇摇头：“没听过。”

提耶利亚等待着下文。

咦？没有了？

提耶利亚在心里嘀咕：喂，你家老爷死了啊，好歹多说几句吧。

\
“咳、咳。那么，是谁第一个发现的尸体？”提耶利亚把眼光抛向众人。

“是我，还有古德曼医生。”说话的是利冯兹。“吃完晚饭后，父亲说有些事务要处理，我便和拉塞扶他回房。他在书桌前坐好后，吩咐我们出房门时将门锁上。我看着拉塞锁上了房门。快九点时，古德曼医生准备离开要和父亲告别，我和他一起上楼敲门，没有回应。我觉得事情不对，便叫来拉塞用钥匙开门，没想到、没想到……”

利冯兹哽咽。

\
洛克昂蹙起眉：“发现尸体的时候，门是锁着的？”他走到窗边检查了一下，窗户从里面上了锁。“那么，这是一起密室杀人案咯。”

利冯兹哼了一声：“所以我才怀疑是超自然生物干的。”

洛克昂不理利冯兹的自言自语，问拉塞道：“卧室的钥匙有几把？”

“两把。一把在我这里，一把还在老爷身上。”

“你今天晚上都在做什么？”提耶利亚问得毫不客气。

明显是被怀疑了，可拉塞的表现还是由内而外的毫无波澜。他平静地回答说：“我在二楼的桥牌室招呼客人们。未曾离开，连厕所都没去过。客人们可以作证。”

“那么钥匙有没有丢失过？会不会被复制了？”

“钥匙从来没有离开我身上。老爷也是一样。”拉塞无比肯定。


\
“这是个很简单的把戏，逃不过我见识广博的眼睛。”冒险者笛卡鲁特发话了：“凶手是把钥匙栓在一根长线上，线头绕过镇长衣服的纽扣。他用钥匙锁上门后，缓缓拉动长线，钥匙顺着线回到镇长的身上后，他剪断了长线将其回收，造成密室杀人的假象。就这么简单。”

提耶利亚心想你是小学生吗，镇长的椅子和门口隔着一张办公桌，桌上成堆的文件，钥匙会随时被勾住导致无法抵达目的地。还没等提耶利亚出声反驳，拉塞便否决了这种操作的可能性：“老爷的钥匙是用环扣拴在腰间的。”

\
提耶利亚检查了伊奥利亚的腰部，钥匙的确栓在环扣上，不旋转两圈是不可能取下的。

洛克昂的目光从众人脸上扫过：

“总之，还是请诸位先说一下，今晚在宴会后都做了些什么吧。”

# 六

洛克昂派拉塞去通知副镇长伊恩·瓦斯提镇长死亡的消息。修亨伯格死后，按照顺位，瓦斯提将成为GN00小镇的代理镇长。这起杀人案件理应由他来主持。

在等候瓦斯提到来的空档，所有人在会客室坐定，讲述他们当晚的行动。


\
除了萨谢斯，其他人在晚饭后都聚集在棋牌室内。利冯兹和亚历杭德罗在棋桌前边喝着拉塞泡的红茶边下国际象棋。因为宴会上喝了不少酒，加上红茶利尿的功效，他们都去过位于三楼走廊尽头的厕所，离开的时间不超过五分钟。

笛卡鲁特、比利和亚瑟一直坐在房间另一侧的麻将桌聊天。比利念念不忘教科书的编纂，而亚瑟则热衷于详细了解笛卡鲁特描述的那些野蛮部落的神奇治疗方法。他们均曾单独离场，时长也不超过五分钟。

萨谢斯声称自己一直在一楼的客厅沙发闭门小憩，没有不在场证明。

“反正不是我杀的。”佣兵扭曲着嘴角笑着说：“我是镇长请来的雇佣兵，如果杀了他，下一个镇长可不一定会继续雇佣我。我没有杀他的动机，你们信不信随意。”


\
提耶利亚从见到萨谢斯那一刻，心里就有个疑问。此刻她终于问了出来：“我很好奇，据说GN00的公职人员都是本地人，镇长为什么要请你来做佣兵？”

比利却抢着回答她的问题：“这件事我听副镇长说过。是这样的，自从上任的两位天使相继死亡后，又发生了两起失踪案件，当时天使组织说还要几个月才能有新天使调任过来，我们镇的护卫队人手一直不足，于是两位镇长决定请佣兵来负责部分夜间巡逻。也神奇了，自从萨谢斯先生来到镇上后，再也没有发生失踪事件。”

“那么新天使到来之后，佣兵也没有存在的必要了吧？”

红发男人脸上闪过一丝轻蔑的神情，他斜睨着提耶利亚，嚣张地说：“不，镇长说让我继续在这里任职至少一年。看来他并不信任你们啊。”

比利赶忙代替已经不会说话的伊奥利亚解释：“啊不不不，绝对不是不信任！我想镇长是想为镇上多添一重保障吧。”。


\
洛克昂没有留意提耶利亚和萨谢斯争执的佣兵去留问题。他用戴着手套的手指抚摸自己曲线分明的下巴，沉思密室之谜：我一直在花园里，这期间没有外人进入这栋宅子。而宅子里面的人，单独离开众人视线的时间都不超过五分钟。假设在没有钥匙的情况下，如何在五分钟之内杀死伊奥利亚并布置密室呢？果然只有魔法才能做到了吧。

\
这时，利冯兹忽然开口打断了他的思考：“请问两位天使大人，这段时间在做什么呢？”

洛克昂直白地回答说在花园赏花，提耶利亚则承认她独自一人待在藏书室内。

“呵呵，看来只有天使大人们没有不在场证明呢。”

利冯兹说得阴阳怪气，意有所指。

\
提耶利亚刚想发怒，手却被洛克昂按住。带着火气转过头，她见洛克昂轻轻摇了摇头，用眼神示意自己别多话。就算心里有100个不甘愿，她也是个恪守规矩的女巫，队长的命令还是要听的。

提耶利亚咽下了差点脱口骂出的话。

只听洛克昂说道：

“不仅我们没有不在场证明，除了拉塞，在坐的每一位都曾经单独行动过，你们的不在场证明都不成立。不管是从一楼还是二楼，走到三楼的卧室杀死修亨伯格镇长，只需要三分钟，你们都有作案嫌疑。”

“喂！我们都没有钥匙，怎么可能在三分钟内进入上锁的房间杀死伊奥利亚再布置密室！”

抗议的是亚历杭德罗，被当作嫌疑犯有失富农继承人的身份，他看起来很不甘心。

“密室手法我会解开的。后续调查就交给天使派出所吧，有什么不满和意见请直接向天使组织举报。我想副镇长也不会有意见吧。”

语毕，洛克昂扭过头，看着刚刚跑进门的伊恩·瓦斯提副镇长。

“啊，当然。归属地的案件侦查本来就是天使大人的义务和权利。”伊恩喘着粗气说。

看来路上拉塞已经把这里的情况转达给他了。提耶利亚有些好奇，就凭拉塞的极简叙事风，伊恩对事态到底能了解到何种程度。


\
涉案人员都是镇上有头有脸的人物，倘若没有证据就随意扣押，会造成不良影响，所以在商议过后，洛克昂和伊恩决定暂时让所有客人先自行回家，在案件彻底解决前，每人每日必须去市政厅报到，并随时配合接受调查。

除了利冯兹之外，其余宾客们都乘着自家马车回到自己的住处。冒险者笛卡鲁特没有自己的马车，他和来时一样，搭着亚瑟·古德曼的顺风车离开了。利冯兹步行回家，他就住在距伊奥利亚宅邸一公里的另一栋房屋内，那是他亲生父亲的遗产。拉塞和伊恩当晚就将镇长的尸首运送至镇火葬场的停尸房冷藏，并封锁了整个宅子。


\
从停尸房回到派出所，已经过了午夜，提耶利亚没想到自己正式成为天使的第一天就摊上了这种事：辖区政府最高官员死亡，自己竟然还有谋杀的嫌疑。

“你累了，休息吧。等下的夜间定点巡逻就由我全部负责吧。我先上去换衣服。”

洛克昂说完这番话，准备往楼上走。提耶利亚在他身后喊道：

“等等。”

“嗯？还有事吗？”

“……真的、不是你吗？”

洛克昂愣住了：“你怀疑是我杀了修亨伯格？”

“晚饭后的一个多小时，你真的一直在花园里？”

“……你不相信我？呵，怎么说我都是你的队友吧。”洛克昂摇了摇头，绿眸子在月光下泛起清冷蓝光。

他凝视提耶利亚，郑重地说：“提耶利亚，我从不说谎。伊奥利亚·修亨伯格不是我杀的。我今天说过这个镇子不普通。可现在看来，对我们来说它甚至是危险的。我们的合作至关重要。我希望你能信任我，可以吗？”

被洛克昂目不转睛地注视着，提耶利亚感到呼吸困难。她不明白为什么会这样。难道吸血鬼的压迫力真有那么大吗？

“好，我相信你。”

语毕，提耶利亚别过头，不敢直视洛克昂的眼睛。她也分不清刚才吐出的这句话，有几分来自胁迫，又有几分出于真心。

# 七

“果然还是钥匙被偷走了更合理吧。”

命案发生后的第二天清晨，提耶利亚和洛克昂对坐在办公桌前讨论案情疑点。

“钥匙在伊奥利亚和拉塞不知情的情况下被复制了，凶手杀死伊奥利亚后再锁上门。能够如此近距离接触他们再复制钥匙的，只有镇长的养子利冯兹。”

昨晚被利冯兹处处针对，提耶利亚本能地讨厌这个绿发少年。

“我看过卧房的钥匙形状，是四重咬合的锁匙，想要复制，不仅需要钥匙模型，还需要门锁的模型。除非利冯兹能把锁匠带到伊奥利亚卧室前……”

“……那还有其他解释吗？”

洛克昂努努嘴：“不如我们先把密室放在一边，看看其他线索。比如凶器。”

“那把武士刀？拉塞说那把刀一直放在伊奥利亚卧室内，凶手是随手选择了它吧……”

“疑点就在这里，如果凶手是有计划杀人，为什么不选择更顺手的凶器，那把武士刀的刀尖没有开过刃，伊奥利亚脑门上的那个伤口需要相当大的力道才能劈出来。”

“一把刀子而已，你抖什么啊？你们巫师从来不玩冷兵器吗？”洛克昂倾身向前，用揶揄的口气慢吞吞地说：“哦，我懂了，你怕我饿了，在你身上割口子喝血。”

提耶利亚浑身上下，最硬的就是那张嘴。

“只有你控制好出汗量，我是不会对你出手的。我不喜欢喝人血。实在饿了就去山上捕猎，这附近的山林里有不少鹿，你要是有兴趣可以一起来，顺便采采药。”

“对啊，每次去打猎都要带上捆木乃伊那么长的绷带。一只鹿吃不饱，喝太多血它就死掉了，给鹿上药缠绷带是件很浪费时间的事。”

提耶利亚暗暗松了口气，心想这吸血鬼也不算太可怕。

“下午我会去伊恩那里了解情况，整晚没睡困死了。上午的巡逻拜托你了，我上去睡觉。”

天气和温度并未因伊奥利亚的死亡和镇民的哀伤染上一丝的阴霾。大气因折射呈现出深浅不一的蓝，大太阳早早悬上半空，肆意释放着浑身散发不完的热量。虽然身边没有吸血鬼，提耶利亚还是不停念着冷却咒语，以免一身臭汗回到派出所被袭击。

\
行经铁匠铺时，提耶利亚特地注意了一下洛克昂把玩的同款小刀，价钱是40个银币。

“好贵，是天使一个月的工资啊……”

吸血鬼擅长聚集财富，这是超自然生物界和人类的共识。这些怪物寿命长、头脑过人、偏爱隐居，喜好在人间囤积珍奇物品，等待个数百年，再以高价卖掉，很少有人能准确得知一个吸血鬼拥有的财富。提耶利亚暗忖她那搭档的年龄，五百岁还是一千岁？都有可能。五百年的岁月无法对吸血鬼的外貌造成肉眼可见的侵蚀。

“咦，这不是女巫大人吗？”

声音有些熟悉，提耶利亚转头一看，是昨晚的客人之一，小镇医生亚瑟·古德曼。他正站在柜台旁看着提耶利亚，满脸的讪笑。

阳光下近距离观察这位医生，提耶利亚觉得用“肥头大耳”形容他再贴切不过了。尤其是下巴上的肉，松弛到好似蘑菇的底部，说话的时候像公鸡的食囊一样上下晃动。

“我是来取定制的手术刀的。镇上只有我一个医生，小到感冒流涕，大到盲肠切除，都要靠我一个人。唉，太累了，真希望什么时候能再来一个医生替我分担一下啊。”

话虽这么说，他的语气倒是满满的自豪，听不出来哪里有疲惫的感觉。

“啊，那祝你早日梦想成真。”提耶利亚用平坦的语调随意应付了一句，准备离开。

“女巫大人请留步！”

提耶利亚眼皮一翻：“还有事吗？”

“昨晚和女巫大人提过，希望大人莅临鄙人医馆不吝赐教，我看择日不如撞日……”

“我还在巡逻中，不能擅离职守。”

“唉，等一下！”

提耶利亚发现自己巫师袍的袖子被古德曼扯住了，她疑惑且不耐烦地看着古德曼：“医生，你到底想说什么？”

“我想问下昨晚的凶杀案……”

“你也是涉案人员之一。很抱歉，恕我不能透露。”

“女巫大人，我曾经读过某部药草学著作，里面有吐真剂的制作方法，据说这种药剂能让喝下它的人说实话，不管他愿不愿意。”

提耶利亚自诩药草学专家，当然听过吐真剂。不止如此，她当场就能背下吐真剂的原材料和制作步骤。

“吐真剂的炼制过程很复杂。嗯，不过对我来说，这点复杂程度算不得什么。困难在于，它需要一种叫七叶红蝎草的植物，这种植物非常罕见。”

亚瑟·古德曼油腻的脸露出神秘的笑容，他低声凑到提耶利亚耳边说：“不瞒女巫大人，我曾经在镇子西边的深山里，见过七叶红蝎草。”

提耶利亚不相信：“真的吗？你没看错？没开花的红蝎草的长相，和鼠尾草很像。”

“每片叶子的叶脉有七道纹路，在火光下会呈现血红色。”

亚瑟说的正是红蝎草的特点。

见提耶利亚表情变了，亚瑟带着谄媚的笑继续说：“女巫大人，我的医馆和您的巡逻路线顺路，我们边走边谈。”

# 八

亚瑟邀请提耶利亚在他下次上山采集草药的时候一同前往，收集七叶红蝎草。提耶利亚答应了。这种用于炼制吐真剂的珍贵草药，在采集后须即刻冷冻、并在一小时内投入炼制，方可发挥功效。能做到这些的，找遍全镇也只有懂得冰冻魔法的提耶利亚自己。

\
“有了吐真剂，就能知道凶手是谁了。哼哼，只靠药剂就能破案，药草学真是伟大。”

完成了巡逻后，提耶利亚开心地乘着扫把返回派出所。她先闻了闻自己身上有没有汗味，确认安全后才敲响洛克昂的房门。她想把吐真剂的事情告诉洛克昂，还有很多准备工作要做，最好马上购置炼制炉和量杯等必需品。

\
她敲了很久的门，房内都无人回应。

“奇怪，他的听觉不是很灵敏吗？难道出去了？”

提耶利亚耸耸肩，她猜想洛克昂大概先去找伊恩了，便在派出所门口的饭馆里解决了午餐，再骑着扫把飞往市政厅。

\
“嗯？洛克昂没来过？！”

得到洛克昂今天根本没来到这里的答复，提耶利亚先是纳闷，继而恼怒。

“明明要我信任他，自己却不把行踪告诉我。……他果然很古怪！”

既然都到了市政厅，提耶利亚便向伊恩询问了当晚几位宾客的详细背景。其中，让她尤为在意的是利冯兹。

利冯兹是在5岁时随生父E.A.雷·阿尔马克搬到GN00小镇的，据说E.A.雷是个国际象棋高手，经常与伊奥利亚相邀下棋，两人以棋为媒成为挚友。在利冯兹十五岁时，E.A.雷因病去世，临死前请伊奥利亚收养利冯兹，把他当作自己的孩子看待。伊奥利亚的太太和女儿在他年轻时相继因病去世，也渴望后继有人，于是答应了好友的请求。后来伊奥利亚腿脚不便，利冯兹主动承担了照顾养父的责任，从生活起居到镇上的政务，他应付起来驾轻就熟。

\
乍一听好像没什么问题。可是，她第一眼见到利冯兹，提耶利亚心里就有异样的感觉，是说不清道不明的厌恶感。

是我想多了吧，也许是我不对？不该因为利冯兹诬陷我就戴着有色眼镜看他。那时他刚刚发现养父被杀，所以一时激动口不择言。提耶利亚自我反省着，强迫自己摁下在心底不断翻涌的“哪里有不对劲”的小问号。


\
提耶利亚一直坐到下午市政厅打烊，都没见洛克昂的影子。马上能制出吐真剂、揪出真凶的喜悦已被抛掷脑后，现在她满脑子都是“洛克昂这家伙到底跑到哪里去了”的疑惑。

他说过这个小镇对我们来说很危险，莫不是……

不会的，吸血鬼不是很强吗？还是担心他会不会控制不住冲动把谁咬了吧。


\
提耶利亚坐着扫把在小镇上空兜了好几圈才回到派出所。她告诉自己这是例行巡逻，眼睛却一直在寻找街上有没有大热天里穿着全套黑色礼服和披风的人影。


\
回到派出所，提耶利亚还没掏出钥匙，门就从屋内打开了。洛克昂神色凝重地从里面走了出来，见提耶利亚站在门外，他“哦”了一声当作招呼。

“我晚上有事。今晚的巡逻就全交给你了。”

交代完这两句话，他便往右边走去，不知要去哪里。

“等等！你去哪！”提耶利亚抓住洛克昂的手臂，脸色语气透着不满和气恼：“另外，我还要知道你今天下午的行踪！”

“我去的地方你不能去。”洛克昂眯着眼，目光顺着提耶利亚的身体，从上到下转了一圈。“至于下午去了哪儿，抱歉，现在也不能对你说，迟一些我一定会告诉你的。记住晚上十二点、两点、四点各有一次巡逻，你上好闹钟，别睡过了。”

他嘴角向两边扯了扯，做出一个似笑非笑的表情，然后轻轻挣开提耶利亚的手，走掉了。

\
呵。

昨晚是谁说过“合作是关键的”？

连行踪都不向队员禀报，这样的队长无疑是失职的！

以为甩开我，就拿你没办法了？

早晚你会因为太小瞧魔法使而付出代价！

\
提耶利亚转身来到派出所后巷，确认四下无人后，念动变形咒语。

\
一只三色猫从后巷蹿了出来，轻巧地跃上对面糕点铺的红砖屋顶，向西一路奔去。

连续跳了十四次房子，三色猫看见目标就在前面不远处，低头吻一位左手捧着花篮的姑娘的右手。猫喉咙咕噜作响，尾巴粗粗地竖起来，弓起背，在斜斜的屋顶做出攻击准备的姿势。

猫看见目标从姑娘手中接过三枝百合花。此时，从南方传来的一股香气直着遛进它的鼻孔，临海的几家鱼肆正在清点刚刚从渔船卸下的新鲜货物。

真麻烦啊，变形之后连这种生物的习性都一并沾染上了。活蹦乱跳的腥气味道提醒着三色猫，它还没吃晚饭。

\
没想到就走了这一会儿神，目标又不见了。猫急得上蹿下跳，干脆跃到地下，从集市人潮各式各样的裤管和鞋底中间穿过，蹿到街对面，往洛克昂行进的方向狂奔。

\
太好了，追上了！猫心里窃喜。眼见洛克昂走进一栋灯火辉煌的四层建筑。

在这个小镇，超出三层的建筑很少见。这是哪里？猫在门外踟蹰了几分钟，决定从后门绕进去。

\
猫钻进后巷。从巷子深处堆着不少厨余垃圾，它猜测这里是个饭馆。

从虚掩的后门挤身而过，进入一间昏黄的、蒸腾着各式香气的大屋子。几个穿着厨师服戴着白色高帽的男人忙忙碌碌，案上除了从镇上唯一一家铁匠铺购买的各式刀子，还堆积着各种哺乳动物、禽类、海中生物分解后的尸身。

若是在变身前，猫绝对不会对这些尸块多看一眼。现在的情况不一样，昔日它最厌恶的肉块在一双瞳孔会自由缩放的猫眼里都变成了珍馐美味。

必须迅速通过才行。猫躲在角落里思忖怎么走才是离开厨房进入大厅的最佳路线。

过道很狭窄，厨师们活动又太频繁，只能从灶台那边穿过。虽然会耗费魔力，这种情况下也只能使用隐身术了。

三色猫在心里默念咒语。噫，它默念的声音在自己听来都是“喵喵喵”。

咒语还是生效了。魔法最重要的不是声音而是心念，只要心念成熟，哪种生物都是可以念咒的，不管是乌龟、蛇还是蝎子，只是效力和持久度取决于念咒生物的魔力值。

\
三色猫在灶台上穿过，它全程屏住呼吸防止自己被尸体碎片的香气勾了魂。跳过一锅开水时，尾巴碰到锅边烧焦了几根毛。

“有什么糊了吗？”一个光头厨师问旁边的同事。

“嗯？啊，烤鸭！”同事慌忙跑到烤炉旁检查。

\
就在厨子们为抢救烤鸭忙成一团的时候，三色猫用爪子扒开门，钻了出去。

猫蹿上二楼，发现有些不对。

等等。

这里不是饭馆吧？

为什么看不见一张桌子？

走廊上只有穿着女仆装的服务员端着酒菜来回穿梭，两边是一扇扇紧闭的门。从门缝里传出的男男女女的声音，在猫的耳朵里听上去有些古怪。

\
还没来得及仔细思考，猫听见一个嗲声嗲气的女人在楼下说：“吸血鬼大人，你要找的人我都叫过来了，她们都在屋里等您呢。”

洛克昂进了三楼某间房。

三色猫跟过去，发现房门紧闭。它费力地维持着隐身，在门缝处缩成一团，还要时刻小心，避免让过往的女仆踩到自己的尾巴，因为剧烈疼痛会让它无法维持隐身魔法。

\
只听门的那一侧有女人的声音响起。

“啊～吸血鬼大人……”

“哈哈哈……”

“您真是的……”

听不太清楚，反正不止一个女人。不久声音就消失了。

猫傻傻地等了一会儿，还是没有声音传出来。

它想了想，门不行，不是还有窗户吗。洛克昂所在的这间房正对着大街，骑着扫把飞行则无法隐身。干脆用猫的形态跳到窗台上去吧。

\
出了大门后，猫解除了隐身状态，跃上旁边较矮民宅的屋顶，再攀着墙壁跳上三楼，沿着一排窗沿，向洛克昂所在的房间优雅踱着猫步。

它看到床上有一个胖子，重重地压在一个女人身上，放眼望去满床都是胖子身上裸露的肥肉，女人露出来的部分只有脑袋，外加一条又白又细的长腿。

被压迫的女人正用全身的力气叫嚷着：“啊！啊！”

猫觉得脑袋一阵晕眩，脚下不稳直直栽到地上，还好在落地前它将身体旋转到位，不是用后背着的地。

猫迅速蹿到后巷。不一会儿，恼怒的提耶利亚从后巷冲了出来，她的巫师袍后摆有一个破洞。

提耶利亚大步迈进刚才那栋楼的大门，往上楼的楼梯方向冲去。没走几步，她被一个脸上浓妆艳抹的中年女人拦住。

“哎呀，这是新上任的女巫大人吗？您到这里做什么啊？”

“我怀疑你这里有犯罪行为，楼上有个男人……正在……让我上去！”

“女巫大人您在说什么啊，我听不懂。”

“我说，你这间旅馆的楼上，正在发生一起强奸案！”

“啊？啊哈哈哈哈哈……”

“女巫大人，您误会了，我们这里是妓院呐，那种事再正常不过了。您放心，上面发生的绝对不是强奸案，每个客人都是事先付钱的。”

提耶利亚悄悄坐在妓院附近小巷的屋顶上，眼光同时扫着天上和地下。这里是回派出所的必经之路。她要质问洛克昂去妓院的目的。

虽然天使的规章里面没有一条明确规定禁止嫖娼，但是第十五条说了：“天使必须品行端正”。

洛克昂出入妓院，怎么看都是品行不端。

她一直等到临近午夜，才见到洛克昂的身影出现在巷口。

从房顶跳下，她凛然站在洛克昂的身前。

洛克昂看见她有些吃惊，他掏出怀表：“还不到十二点你就出来巡逻了？工作挺勤快吗。”

“不是。我一直在这等你。”

“等我？等我干什么？”

“你刚才去哪了？”

“不是和你说了吗，时机到了我会告诉你。”

“你去妓院了！”提耶利亚掩盖不住恼怒，虽然她跳下来前千叮万嘱要自己冷静。

“你跟踪我？”洛克昂皱了皱眉，旋即不在乎地耸耸肩，“算了，我原谅你的行为，下次不要这么做了。你继续巡逻吧，我先回去了。”

洛克昂从提耶利亚身侧掠过，继续走他的路。

\
“站住！”

提耶利亚在身后喊。

洛克昂没回头，他向后摆摆手，打了个哈欠：“我累了，有什么事明天再说。”

> 早晚你会因为太小瞧魔法使而付出代价！

提耶利亚忘记了咒语老师在课堂上的谆谆教导，掏出魔杖对洛克昂的背影念出了定身咒语。



“叮、叮叮、叮叮当。”

巷子里响起几声金属和石板地碰撞的声音。

洛克昂站住不动了。

然而，他不动的原因并不是定身咒语生效，而是他身上所有的布料，都消失不见了。

而纽扣、怀表、别针、领巾扣、腰带环这类饰物，跌落了一地。

月光下，吸血鬼像石膏一样白的裸体看上去极富艺术欣赏性。

提耶利亚愣了两秒钟，想起此时应该捂眼大叫才对：

“你、你为什么突然脱衣服！这是何等失态！”

“别叫了！”她闭着眼，听见洛克昂恼羞成怒的声音：“不是我脱的，我也不知道为什么会这样！快把你的袍子脱下来借给我！”

# 九

背对着洛克昂，提耶利亚脱下了巫师袍，取出收纳在袍子口袋中缩小的飞行扫把和其他零碎物件。

临近深夜，巷口那条街依旧热闹。提耶利亚生怕有人进来撞见他俩此时此刻难以解释的状态。她将挂着巫师袍的手臂向后甩了甩，指尖从洛克昂冰凉的后背上扫过。

\
“你刚才在我背后念叨的是什么？”

用宽大的袍子遮住身体后，洛克昂转过身质问。

“啊……那个……”提耶利亚正犹豫应不应该说实话，远处镇政府楼顶的大钟及时响起来。

“唉呀，午夜到了，我该去巡逻了。你快回派出所吧，刚才在妓院里折腾那么久一定累了……”

提耶利亚急急骑上扫把，闪电般“嗖”地飞上漆黑的夜空，速度太快以致于转弯的时候身体晃了一下，也因为速度太快，她没看见身后巷子里的洛克昂嗅着巫师袍衣领、那皱起眉头的样子。


\
提耶利亚在镇子上空盘旋了40多分钟后回到了派出所。为尽量避免发出声响，提耶利亚将钥匙缓缓插入大门，轻轻旋转，踮着脚尖滑进门缝，仅用脚掌着地朝楼梯的方向挪动。

“啪”一声响，吊灯亮了。

吸血鬼还穿着那套固有的黑色礼服和披风，优雅地靠着椅背。他脸上挂着迷人的微笑，对双脚只有一半着地的提耶利亚说了句：“辛苦啦”。

桌子上放着提耶利亚的巫师袍，袍子底部因热锅灼烧产生的破洞被翻到了正中间。

“多谢你的袍子，否则今晚镇上夜行的人将会看到一个裸体吸血鬼在天上飞。”

洛克昂把巫师袍往前一推，随后按住提耶利亚欲取走袍子的手。

“现在能告诉我了吧，你到底在我身后念了什么？别想否认，我听得很清楚。为什么我的衣服会忽然消失？”

在洛克昂凌厉的眼神逼问下，提耶利亚只好说实话：

“我对你施的只是定身咒，因为你不肯停下来嘛……我忘记巫师的咒语在吸血鬼身上的效果是混乱的……不过，混乱效果大概也和吸血鬼本身的品行有关。对！一定是你去了那种地方，所以咒语才让你的衣服凭空消失。总之，会变成这样你自己也有责任……还好没有变成夺命咒的效果，能保住一条命就应该庆幸了！”

提耶利亚越说越觉得自己夸大其辞的话很有道理，她甚至有些兴奋了。首先，她的咒语在吸血鬼身上是生效的，且并非反弹效果，没有对自己造成任何伤害，而大部分巫师的咒语在吸血鬼面前是完全无效的。其次，“裸身”的效果，虽然不会造成物理伤害，但心理伤害总有吧，洛克昂是个要面子的人，她的咒语对他至少有些威慑作用。

想到这里，提耶利亚嘴角微微上扬。

“啊？别把你的鬼咒语赖在我头上！我到妓院是查案去的，什么都没做……哦呵呵，我懂了，会造成那种效果，一定是因为你觊觎我的身子很久了吧。啧，下次你想看直接到我房间就好啦，千万别再使这种阴招。我总共只带了十套衣服过来，全都是巴斯德市最好的裁缝那里定制的，预约都要两年呢，玩不起你的情趣play……”

洛克昂巧舌如簧，几句话就把“裸体”的锅反扣到了提耶利亚的头上。

“谁、谁觊觎你啦！你不要血口喷人！”提耶利亚脸憋得通红，怒气让她的肩膀和胸口起伏不断。

洛克昂的目光向下移，揶揄的口气中还带点惊奇：“欸，平时一直用巫师袍遮着，想不到你身材还不错嘛。”

为了减少出汗量，提耶利亚巫师袍的内部只穿了一件薄纱鹅黄衬衣。在吊灯的光线下，内衣的轮廓隐约可见。发现洛克昂正在看什么之后，提耶利亚羞恼地抓过桌上的巫师袍披回身上。


\
就算被污蔑自己对洛克昂有非分之想，提耶利亚还是没有漏掉他话语中的重点：“你刚才说’查案’？妓院难道和凶杀案有关吗！为为什么不事先和我说？！”

“啊，说漏嘴了。”洛克昂显得有些为难。

“哼，你一五一十告诉我吧，否则我让你身上这套衣服也消失掉……”

说着，提耶利亚举起法杖。

“好，我告诉你。这套衣服是我最喜欢的，你千万别念咒。”


\
等提耶利亚在对面的椅子上坐定，洛克昂开始了他的讲述：

“事实上，在我来这个小镇上任之前，我那个吸血鬼师父——九大天使之一的力天使——交给我一个任务，他怀疑我们前任的那两个巫师天使，是死于魔法。”

“魔法！？”提耶利亚难掩震惊，“你是指有其他巫师杀了他们吗？难道是巫师的内斗？不可能……”

否定的气势明显弱于提耶利亚的正常阈值，因为她心里开始盘算巫师界近数百年暗涌不断的内斗，数名位高权重的大巫师莫名其妙死亡，巫师协会验尸后给出的结论有“钻研魔咒走火入魔”、“脑中风”及“来自恶魔界的诅咒”等等，且报告内容模棱两可，无法使人信服。

洛克昂接着说：“来这儿之前，我详细研究了那两个巫师留下来的保管在天使组织总部的巡逻日记，发现他们相继死亡的前一个月，两人正在调查镇上几年前的一起人口失踪案件。根据他们的笔录，失踪的是个妓女，而在妓女的房间里，他们找到了施展魔法咒语的痕迹。”

提耶利亚摇头道：“哼，会留下痕迹，一定是个低级巫师的杰作，低级巫师又怎么可能杀得了天使？”

“这点就不得而知了。继续调查的过程中，两人离奇死亡，不过笔录的最后部分，他们提到了准备彻查近年来发生的另外两名妓女失踪案件，于是我就向妈妈桑寻求帮助，让我和三位失踪女性相熟的朋友们见个面。”

“那你今晚就是去见她们了？”

“是啊。”

“问个话需要一整晚吗？”

“呵，其中有一位姑娘来自西部。她听说我会飞后，让我把一封信和钱交给她故乡的父母，一个来回耗掉两个小时，还要加上等她父母回信的时间。”

“哦，原来是这样……看不出来你还挺温柔。”

“嗯？‘看不出来’几个字是多余的。”

“……那你的问话，查出什么没有？”

洛克昂迟疑了一下，摇头道：“暂时没有明确的目标，三个人的熟客没有交集。”

“哈，也就是你做了一晚上的免费信差，什么都没查到？”提耶利亚的言语夹杂着得意地讥讽：“以后如果再有这种调查，记得叫上我，也许我能发现你看不见的盲点。”

洛克昂失笑：“喂，那里是妓院啊，不允许女人进入的。”

“我会形意术，可以变成男人。”

“哦？”洛克昂好奇了，“我还是头一次听说巫师能变性。你现在能变一个让我看一下吗？”

“小菜一碟。”

\
提耶利亚昂首起身，念动形意咒。感觉身体起了变化后，他走近洛克昂说：“好了。”

“嗯？”

“好了。我现在是男人了。”

洛克昂上下打量着提耶利亚，两道浓眉蹙到一起：“你哪里是男人了？”

提耶利亚恼怒地抓起洛克昂的手，放到自己胸前：“我的身体是男人啊！”

来回仔细确认后，洛克昂哈哈大笑起来。

“喂，你可真了不起，原来变成男人就是把胸缩回去啊！你的脸还和做女人的时候一样，这个样子能糊弄到谁啊！别说镇上的人都认识你的脸，就算没见过你的人，第一眼看上去也会认定你是女人吧。不行不行，不能带你，不管哪儿的餐馆都是禁止外带食物的啊。”

\
继咒语被嘲笑后，形意术又被奚落了一番，自己还被比喻成了“食物”。提耶利亚几乎是强压着再次让洛克昂的衣服消失的冲动，喘了口粗气，重重咳了一声，说：“总之，你以后的一切行为都要通知我。如果再出现擅自行动的情况，我就在你背后念咒！”

洛克昂止住笑，点点头说：“好、好，我知道了，真是怕了你了。我会提前告诉你我的行动，你也千万别冲动。”

# 十

第二日午间，洛克昂独自外出巡逻归来，手上多了两把一人高的大铁锹。他潇洒地把铁锹往派出所墙角一扔，响声惊动了正专心阅读着上一任天使留下资料的提耶利亚。

\
提耶利亚瞥了一眼，说：“拿那玩意做什么？要改行种地吗？”

漫不经心地问了这么一句后，她的眼睛又回到了手里的卷宗上。

\
洛克昂整理了一下领巾，回答说：“今晚我们去开棺验尸。”

“啊？！”提耶利亚合上卷宗，瞪着眼睛问：“开谁的棺？伊奥利亚还没下葬呢。”

“他的遗体已经找不到新线索了。我们去开两个前任天使的棺材。镇民认定他们是意外和病故，所以死后尸体没有经过任何检查就直接下葬了。只埋了几个月，应该还没腐烂，说不定会有新发现。下午我会去趟市政厅，找镇长申请一份掘墓许可。”

\
提耶利亚微蹙眉头，像是对什么犹豫不绝。

刚才外出就餐时，她碰见了亚瑟·古德曼。

说“碰见”并不恰当，古德曼医生是专程来找她的。他说今日会去西山采药，按照上次约定的，请提耶利亚在日落后到山下的石碑处等他。

七叶红蝎草必须是于黑暗中、火光照耀下才能与其他杂草区分开来，因此只能在夜间采集。

相较可能是由于巫师内斗引发的前任天使之死，提耶利亚还是更愿意把时间花在调查镇长被杀的案子上。一来碍于巫族的身份，她不愿与“内部争斗”扯上关系，二来查清镇长案子的功劳更容易获得镇民的尊重与爱戴，与她上任时给自己制定的目标——获得更多的人类爱戴之情——是相吻合的。

提耶利亚在心里打定主意。她咳了一声，郑重地说：“嗯……洛克昂，你看这样如何。你去全力调查力天使交给你的案子，伊奥利亚的案件就交给我。我们分头行动，不要因为拘泥于团队协作错过办案的最佳时机。”

洛克昂做沉思状。他的手指摩挲着下巴，对提耶利亚的提案提出异议：“这样好吗？如果真有其他不怀好意的超自然生物曾在这个镇子出没，他们可能会再次现身。倘若在单独行动中遇到危险……”

提耶利亚思忖洛克昂的担心不无道理。假设能够杀死天使的超自然生物真的存在，她可没有单独挑战它们的信心。她缺乏和其他超自然生物战斗的经验。巫师学院考试时的模拟战在实战时能有多大帮助，提耶利亚一点儿底都没有。

\
“啊！我有办法了！”

她突然灵光乍现似的，一溜烟跑到楼上。片刻，她拿着两个金色哨子下了楼。

“我对着这两个哨子施了同步咒语，只要一个哨子响起，另一个哨子便会同步震动。距离越近，震动越强。你接着。如果我们单独行动的时候遇到危险，就吹响哨子作为信号。”

提耶利亚吹响了手中金哨，洛克昂感到他手中的哨子嗡嗡大震，手心发麻。

他盯着手里活蹦乱跳的哨子，将信将疑地问：“你的魔法持续的时间有多久？不会突然失效吧。”

提耶利亚自信满满：“当然不会。除非我再次施法解除咒语。”

\
洛克昂从伊恩那里拿到了掘墓许可。除了许可文书，他还额外带了个青年一起回到派出所。

这青年名叫哈罗，是镇护卫二队的小队长，号称护卫队里最机灵的人物。

伊恩将政府全权许可令交给哈罗，嘱咐哈罗在案件得出结论之前协助洛克昂办案。因为哈罗是人类，不会飞行，怕他跟着洛克昂在行动速度上会拖后腿，伊恩还特别批给他一辆政府的公用马车。

哈罗二十岁上下的年纪，穿着一身护卫队制服向提耶利亚行了个礼。世界上没有比“圆形”更适合形容他长相的词语了。脸型是正圆的，眼睛是椭圆的，鼻翼是双圆，耳朵是半圆。脸虽然圆，身体倒挺壮实，身材与洛克昂一般高大，虬结的肌肉虽比不过拉塞，也相当可观，隔着制服能依稀看出胸肌的轮廓。他那一对长长的手臂尤为扎眼，自然垂下时，双手几乎能碰到膝盖。圆滚滚的长相和成熟的身型极不协调，好似一个男版的“金刚芭比”。

\
昨晚，提耶利亚要求洛克昂将一切行动提前告知她，洛克昂今天便这么做了。向提耶利亚报告了公墓位置和预计行动耗时后，洛克昂让哈罗抬上铁锹，两人一同出发去往坟场。

洛克昂走后，提耶利亚也开始为炼制“吐真剂”做初步的准备。她把挖药草的银质耙子、小镰刀、存储草药的耐冷冻水晶瓶放进皮箱内，又把她日前购买的各式蒸馏器皿清洗干净，在她房间里架好炼制炉，并将其他材料按照七人份提前称量好。利冯兹、亚历杭德罗、萨谢斯、笛卡鲁特、比利、亚瑟、拉塞——提耶利亚很公平地为每个嫌疑人都预留了一份可以让他们“洗脱嫌疑”的药品。

\
傍晚，按照约定时间，提耶利亚飞到指定地点。她在天上远远看见亚瑟站在石碑前，焦躁地搓手等待。提耶利亚在他面前落地后，他脸上的褶子随着笑容像烟花一样绽开。

“女巫大人，你总算来了。我已经恭候多时了。”

提耶利亚冷淡地点点头，“嗯，天马上黑了，我们抓紧时间。”

\
亚瑟提着一盏古铜油灯走在前面，提耶利亚跟在他身后。他们先沿着一段人工铺就的石子路走了一段，步行到某个岔路口，亚瑟停下脚步，举起油灯观察周围的树木。端详了足足两分钟后，他指着右边的路说：

“请走这边，女巫大人。”

\
接下来的两个岔路口，亚瑟都显得不是很确定。这时，提耶利亚对于此人是否能帮她找到想要的东西产生了怀疑。

再后来，他们逐渐偏离了攀山的主路线，走进铺着厚厚落叶、每踩一步都要分外小心才能防止跌倒的山林内。夏季蚊虫数量多，蛾子在古德曼的油灯四周扑棱棱地飞着。提耶利亚心中默念驱虫咒，问古德曼：

“古德曼医生，我很好奇，此处荒无人烟、地处偏远，你是因何契机发现这山里有七叶红蝎草的？”

“啊，女巫大人，我猜您一定会问我这个问题的。说起这故事，还是一段奇闻呢……”

\
提耶利亚后悔了。她这时方才记起，古德曼在伊奥利亚的晚宴上是如何向她展现口若悬河的演说技巧的。

医生跳过了他的童年经历，从年少去外地拜师学医的机缘开始讲起，讲了他与当今世上几位名医的师从关系，再讲到他求学过程中是如何刻苦钻研草药、学成归来后又如何用所学知识悬壶济世、造福一方的。

几年前，他得到一本巫医术的药草学医典，对这门学问产生了浓厚兴趣，然而小镇地处偏远，苦于无法购买医典中提及的各种珍奇花草。没想到某天在镇西边碰到一位樵夫，发现他砍的柴中竟混杂着一株七叶红蝎草，方知这西山乃是一方宝地。此后他为替镇民谋福祉，常常自己登山采药，蒙上天眷顾，让他在此过程中找到了七叶红蝎草的所在地。


\
提耶利亚被亚瑟长篇大论的学医史搞到昏昏欲睡。若是头脑清醒时，她一定不会放过这个纰漏：一个山野樵夫好好地砍柴，怎么可能砍到七叶红蝎草这种四周经常有毒蝎出没的危险植株。


\
“哎呀，就在前面了！女巫大人，看见没？在那边、红色的，就是你要找的东西！”

在油灯火光照射下，提耶利亚看见前方草丛中隐隐闪着幽幽红光，的确有点像七叶红蝎草的颜色。

“女巫大人，我怕毒蝎，不敢上前。这油灯给您，恕在下无法奉陪。”

“好，辛苦你了。”

这种在巫师领地都难得一见的珍贵草药让提耶利亚心里激动不已，但她绝不会把这种激动在人类面前表露出来。平静接过亚瑟手上的油灯，出声念起驱虫咒（为了对付魔力比人类都强大的毒蝎，必须如此），提耶利亚朝着前方的红光走去。


\
她没能发现红光是魔法造出的假象。距离真相只有三步之遥时，她的脚底忽地踩空，跌入了陷阱中。

# 十一

提耶利亚重重摔进了一个足足有三米深的坑里，右肩先着的地，跟着是脑袋。坑里的空气潮湿的像油一样，她眼冒金星，脑部因撞击产生了短暂眩晕。在能够思考发生了什么之前，嗅觉先醒了过来，一股奇异的香气蹿入她的鼻孔，面部的毛孔像打了麻药一样，渐渐丧失了触觉。

\
中招了。这个人想要杀我。

提耶利亚的大脑恢复了活力，只花了一秒钟便机警地读懂了眼前的状况——她的皮肤正在丧失触觉，撑住地面的手已经感觉不到地上的沙砾，接着，手臂的力量像是被抽走一般突然消失。

周围空气里弥漫的怪香一定是麻药，而且是混入了魔法的麻药。先闭起呼吸减少摄入……然后呢？啊，洛克昂！哨子！

\
以狗啃泥的姿势伏在地上，提耶利亚勉强抬起颤颤巍巍的手，从领口抽出了挂在脖子上的金哨子。她把它含进嘴里，对着吹口吐了口气。哨子静悄悄的，并没有出响。因为这家伙不久前才被施了法术，正春风得意呢。想要它吭声，需要空气动力比正常值要大出许多。

提耶利亚没有放弃，她从不是个轻易放弃的人，深吸一口混杂着麻药的空气后，又尝试吹第二次。

这次更加不幸，她连将嘴唇撅到一起的力量，都没有了。

\
亚瑟·古德曼不知何时带上了防毒气的猪鼻子面具。虽然此人已被证实属于敌方阵营，提耶利亚还是要承认面具和那张脸是一对绝妙搭配。他站在陷阱边缘，冲着下方的提耶利亚呵呵笑着。

“女巫大人，现在感觉如何？啧啧啧，之前每次见你，都是一副高高在上斜眼看人的样子。偶尔躺在井底向上看，景色也别有风味嘛，你说对不对。”

他用向上的语调结束了这段话，最后一句应该是个问句。不过他心里没期望回答，所以，也可能是个反问句。

提耶利亚怎么可能回答呢。她四肢瘫软地趴着坑里，气若游丝，放任从坑壁四围不断涌出的异香顺着呼吸道侵犯到到肺部，再顺着血管游走至全身各处。

现在的她，已没有了物理杀伤力，这是废话。另外，迷药里还混杂着强大的巫术，这种巫术让她的魔力也一并消去了。提耶利亚后悔了。洛克昂说的没错，这个小镇上存在着第三个超自然生物，但那人并不是亚瑟·古德曼——就算再怎么暴饮暴食，超自然生物也不可能长得这样胖。他只是被操纵的傀儡。


\
“女巫大人，”亚瑟侧着脑袋，一边欣赏提耶利亚翻白眼的目光，一边咧嘴笑着继续说道：“你知道为什么我对镇上的人有这么大的贡献，却从不居功自傲，总是谦谦有礼吗？因为我都觉得，傲慢是很坏的东西。它不但是原罪，更是一种疾病。我尤其痛恨患了这种病的女人，她们最需要被治疗。不瞒你说，我第一眼见到你的时候，就诊断出你病得不轻——眼睛长在头顶上，说起话像全世界都欠你的钱一样。不过，幸好还没到无可救药的地步。你放心，经过我给你量身定做的几个疗程，你一定会痊愈。唉，只可惜，就算痊愈，你也活不了多久了。可能这就是人she——”

在亚瑟·古德曼演讲的期间，提耶利亚全场愤懑地斜睨着他，尝试用眼神杀死这只猪猡。也许是眼神的杀伤力起了效果，“生”字仅发出了一半的声音，亚瑟忽然从井边消失了。


\
提耶利亚正纳闷怎么回事，一个熟悉的嗓音在上方响了起来。

“没事，他只是晕了。哈罗，你身上有手铐吗？把他拷起来。此人涉嫌谋害天使，先押回监狱吧……他这么沉，你一个人扛得动……哦，还真厉害……好，明天见。”

是洛克昂。

提耶利亚知道自己会被耻笑羞辱一番，但无所谓了。洛克昂的声音将适才心里的恐惧一吹而散。现在他对她讲什么话都可以。


\
洛克昂的嬉皮笑脸出现在了井的上方。

——“喂，你今晚就睡这里吗？这儿好像有很多蚊子啊。”

提耶利亚想要否定。可是她说不出话，她连动动嘴皮子的力气都没有了。

“我把你拉上来。”

洛克昂勾了勾手指。

这是他们初次见面时就施展过的雕虫小技，吸血鬼专属的超自然能力。提耶利亚的身体像一团胡乱包着的棉花，从井底飞了上来。

那个随身皮箱也一并上来，撞在地上，银质的工具跌了一地，在夜色下隐隐闪着黯淡幽光。

洛克昂好奇地蹲下查看：“这都是什么啊？噗……你不是说你不采药的吗？莫非你大半夜的，是来山上锄草种地？”

洛克昂嘴上揶揄个不停，手上还是帮她把镰刀、耙子、玻璃瓶一件件捡回箱子放好。


\
躺在地上的提耶利亚大口喘着气，一句反驳的“呸”字都说不出来。她脱离了陷阱，不再吸入怪异的香气，魔力也渐渐恢复，本以为只要施展解毒咒语就能消除迷药的效果。谁知默念了几遍解毒咒，非但身体状况毫无起色，反而急火攻心，皮肤渐渐发起烫来。


\
洛克昂本想接着取笑她几句的，可是发现她状况不太对便作罢了。他取下手套，摸了下提耶利亚的额头。

此时，身体仿佛放置在熊熊烈火的边缘的提耶利亚，只觉得下一秒就要投入焚化炉烧起来。洛克昂凉血生物特有的冰凉指尖贴到额上肌肤的那一瞬间，她感到异常舒适，恨不得这几根指头能就此粘在自己脑门上。很遗憾，洛克昂试了下温度就收回手指，戴上手套。


\
“好烫……总之，先送你回去吧。”

洛克昂先把放有提耶利亚“种地工具”的皮箱背在身后，再一手插到她的膝下，一手揽住她的后背，毫不费力地将提耶利亚横着抱了起来，像抱着一团棉花一样轻松。

她的头抵在洛克昂脖子和肩膀的交界处。洛克昂马上发现了这是个多么糟糕的姿势——提耶利亚的头发就在他鼻子的正下方，她的身体是绵软滚烫的，散发出的特殊气味正不停刺激和煽动着他的食欲以及性欲。


\
洛克昂唯一能做的就是加快飞行的速度，让擦身而过的飞速气流稀释气味的浓度，再尽量减少自己呼吸的次数。

等把这团棉花扔到她自己的床上时，洛克昂已经处于食欲的临界点了。

躺在床上的提耶利亚胸口上下起伏，呼吸依旧粗重，好像鼻子嘴巴共同不懈努力都无法填满她的肺活量，面颊是不正常的烧红色，嘴唇也快皲裂了。

空气里弥漫着喘息的声音和诱惑的气味。洛克昂强忍着巨大的生理冲动，给提耶利亚倒了一杯水，并喂她喝了下去。

提耶利亚贴着他的身体，费力地咽着杯子里的凉水。洛克昂的嘴也没闲着，不断吞下口腔因巴浦洛夫效应产生的口水。他清楚听见提耶利亚的心跳和自己的心跳在胸腔交替鼓动的噪音，二者同样的紊乱。


\
只要能马上逃离这个弥漫着诱惑气味的房间就没事了，莫冲动，要冷静——洛克昂这样提醒自己。于是他转过头不看提耶利亚，努力让五感远离使他内心骚动的源头。


\
提耶利亚咽下整杯水后，呼吸平稳了许多。洛克昂不敢问她有没有再喝一杯的需求，因为他没有信心能喂完第二杯。他把空杯子撂在桌上，迫不及待向着门口冲去。


\
身后那张床上传来的细若蚊蝇的声音：

“你……别走。”


\
洛克昂站住了。

一头三天没进食饿得前胸贴后背的狮子，会拒绝一只在它身边蹭来蹭去的温顺羚羊吗？

呵呵，当然不会。

食色性也。

洛克昂回到床边，凑近提耶利亚的脸，俯身在她耳边低声说：

“提耶利亚，是你让我留下来的……现在后悔还……不，已经来不及了。”

# 十二

地上散落着的衣服乱糟糟的，堆得像一座塔，由下至上依次是洛克昂的披风，提耶利亚的斗篷，提耶利亚崩掉纽扣的衬衣，提耶利亚的长裤，提耶利亚的内衣，洛克昂的上衣，洛克昂的长裤……

\
洛克昂的左手的手指在提耶利亚的胸部浑圆的部位徘徊，右手滑进提耶利亚的内裤。冰凉的指尖碰到发烫的地带时，提耶利亚像受惊的兔子一样打了个寒噤，跟着猛吸了一口气。

\
“怎么，怕了？”

提耶利亚没有回答洛克昂的问题。她半闭着眼，头扭到一边。

现在的她不像十分钟前那样燥动了。至少头脑冷静下来了。这要归功于适才与眼前的凉血生物裸身相拥，还有——放血。


\
传说中人类初吻的味道是草莓的香甜。提耶利亚的初吻是鲜血的腥甜。

刚才，洛克昂用近乎撕扯的方式脱掉了她身上大部分的衣物，手指插进后脑的发丝，紧紧压实她的双唇。吸血鬼的尖牙从干燥欲裂的唇瓣轻轻划过，血渗了出来，来不及凝成血滴，就被洛克昂的舌尖舔舐得无影无踪。

嘴唇的伤口很快止了血，显然，仅凭它们满足不了洛克昂的胃口。他的下一个目标，是提耶利亚葱白的纤纤十指。

他从食指开始品尝。洛克昂慢慢将提耶利亚整根手指吞入口内，她的指尖抵到他的上颚，然后用柔软的舌头打着转吸吮。待到她的手指被冰凉的唾液浸泡完全，再咬开指尖前端，用双唇含住，将溢出的鲜血吸入口腔。

提耶利亚左手的四根手指都遭受了此等虐待。也许是她中了迷药的关系，也许洛克昂的唾液有止痛作用，她没有感到痛。她眼神迷离，看着身体上方的洛克昂对她做出这样那样的动作。洛克昂也回看她。四目交汇时，提耶利亚的心跳快到让她慌乱不已。


\
指尖血终于让洛克昂的嗜血欲望得到了些许缓解。他再次封住提耶利亚的嘴，两人进行了一次像模像样的长吻。提耶利亚的木讷和笨拙很快融化在对方冷静且柔软的双唇和舌头间的缝隙里。不过，让她感到更为受用的，是两具赤裸躯干的相互贴紧。他们贴得很紧，紧到一方身上纤细的汗毛可以钻进对方的肌肤纹理。洛克昂冷感肌肤对于浑身灼热难耐、如同在炭上炙烤的提耶利亚来说，无疑是是炭中送雪。


\
提耶利亚的清醒没有维持很久，洛克昂的爱抚让她刚刚回来的思绪又飞到云里雾里。下体渗出的爱液浸湿内裤，潮湿的布料贴着洛克昂的指节，也贴着她的皮肤。指尖在凸起和凹陷之间蠢动，她的心也一样。


\
“你是第一次？”

洛克昂的中指的第一指节探入后这样问。

提耶利亚没有用“是”或“否”来回答，她咬着下唇，下颚上下抖动了一下。

洛克昂微笑。他将中指抽出，然后撕破了提耶利亚内裤的蕾丝花边。动作干净利落，就像我们撕破食品包装袋的手势。他的食指和中指毫不犹豫地捅破了阻碍他前进的路障，就像我们用吸管戳破牛奶盒一样。

洛克昂不喜欢浪费。他舔干净手指，然后吸吮从提耶利亚下体渗出的血液。那血液混杂着欲望的味道，美味极了。

提耶利亚还是没有感觉到痛，某种更强烈的刺激掩盖了痛觉。洛克昂的舌尖在她体内翻搅，她感到一股电流从尾骨沿着脊柱蹿上头顶，跟着浑身一阵酥麻，她叫出了声音。她的双脚在床单上来回摩擦，大腿内侧被洛克昂的发梢撩得发痒。洛克昂停下看了她一眼，两手固定住她的双腿，让它们分得更开，然后继续。

他一直等到提耶利亚呻吟的音量让他满意后，才将等待了很久的分身挺进她的体内。探入的过程，提耶利亚一直猛烈地摇着头，伴着介于呻吟与尖叫之间的喊声。洛克昂压住她的身体，她的指甲几乎嵌进他的肩胛。

完全进入后，洛克昂停了下来。提耶利亚疑惑地睁开眼，看见洛克昂也在看她，他的眼角正温柔地笑着。提耶利亚心脏一紧，问：“怎么了？”

“没什么。只是觉得你的样子很可爱，声音也是。”

提耶利亚感到面颊倏然升温。

“你胡说什么。”

她的声音低低的。不是问句，也不是陈述句。

洛克昂把提耶利亚歪到一边的下巴扶正，对着嘴唇吻上去，吻到她浑身发软，吻到她能够适应留在体内的硬度和粗度。

洛克昂稍稍抬起上身，开始冲撞。起初的力道很浅，而后渐渐加重。他们的喘息也在加重。两人借由交合和拥抱重叠在一起，洛克昂那部分是冰，提耶利亚那部分是火，他们注定要为对方变形、溶解、熄灭。

提耶利亚觉得自己是一艘小船，航行在波涛汹涌的海上，一阵又一阵的海浪袭来，让她晕眩、迷失。洛克昂的拥抱就是海，她的灵魂在那里面翻滚、触礁，沉溺到底。与其说是海浪掀翻了她的船，不如说海底有什么在吸引她，是暗涌的漩涡拉扯着她引她下沉，而她亦懒得挣扎。她放任洛克昂在她的身上为非作歹，甚至他的尖牙几次抵上她的颈部外动脉又离开，她都没有反抗。此时此刻，提耶利亚是无所谓的。事后她感到后怕，肉体极大的快乐甚至令她丧失了求生本能。

善于统计和推算的提耶利亚并未对她初夜的高潮次数进行计数。到天亮前，大概连“数字”这个概念，都从她的头颅内消失了。

两个天使整晚都没有巡逻，第二天他们都写了检讨书，原因一栏，提耶利亚填着“中毒”，洛克昂填着“同事中毒，需要照顾”。

# 十三

他们彻夜未眠。天亮后，洛克昂的下巴抵在提耶利亚的肩窝上，问她刚才发生的一切会对他们今后的共事产生什么影响。提耶利亚不语。她推开身后的吸血鬼，起身取出一套新的内衣换上。她看了看地上的衣物，昨天摔进陷阱后它们沾上了泥，不能穿了，于是从衣柜拿出干净的衬衣和长裤，有条不紊一件件套到身上。遮盖了所有不该裸露在外的肌肤后，提耶利亚转过身，正视着手肘支在床上、盯着她做完一系列动作的洛克昂，回答说：

“没有影响。昨晚的事是意外。我中了毒，需要你的帮助。以后不会再发生了。我们巫师间有句话，「只发生一次的事情等于无」，用来形容魔法的意外效果。我想这句话也可以描述昨晚——只有一次、不可能重复，等同于什么都没发生。我去洗漱，你快回自己房间吧。”

洛克昂一脸困惑，似乎不太理解提耶利亚话是什么意思。他低声自言自语道：“什么嘛。只发生一次？昨晚可不止一次吧……到底是几次来着？”

\
之后洛克昂前往护卫队提审亚瑟·古德曼。提耶利亚本想同去，洛克昂用“你是受害者，不能参与审案”为理由拒绝了。

“嗯，你说的有道理。”提耶利亚点点头，没有坚持自己的意见。

\
提耶利亚独自走在，街上执行巡逻任务。她的眼光游离地四处张望，心里惴惴不安。镇子看上去很正常。阳光依旧灿烂，海浪依旧欢腾，集市依旧熙熙攘攘，人们如常向拥有天使身份的她点头致敬。和昨天不一样的，是提耶利亚的心情。整个早晨她都在佯装镇定，可是发生了那种事，让她怎么镇定得下来。她是女巫，也是“天使”，作为一个超自然生物被人类欺骗已经足够沦为笑柄了，被吸血鬼救了更是惨上加惨，最后竟然还和吸血鬼发生了那种关系……

她也不是天真到什么都不懂。虽然没有实战经验，通过阅读与人体（超自然生物和人类有相同的身体构造）相关的书籍，她对那件事还是有些常识的。不过书本上模糊的描述不够她界定昨晚身体产生的强烈感觉是由于药物反应、还是咒语出了岔子，又或者是本该如此。

她说了谎。什么“没有影响”、“纯属意外”，都是谎言。她现在看洛克昂的眼神都变了。一看到他的脸就会心律紊乱呼吸急促——类似心肌梗塞的症状——需要默念凝神咒才能缓解身体的躁动。一定是做了那种事情的创伤后遗症。好在掩饰的效果不错，至少洛克昂没发现——好像是没发现。

能彻底抹去这段记忆就好了。她倒是很愿意忘记昨晚的事来着，可惜失忆咒不能施加给自身，况且这个咒语副作用和不确定性太大，万一没有忘记该忘的，把咒语和药草学理论忘记了如何是好。这个方法行不通。


能不能申请调职，离开洛克昂呢？看不见他就什么都解决了。虽然有五年的限制，可是凡事总有例外吧。找个什么借口呢……提耶利亚绞尽脑汁，也思索不出一个既合情合理又不会被人耻笑的理由。

提耶利亚心不在焉地走啊走，行经到镇西南的花店处，她感到挂在胸口的金哨子震了起来。

“洛克昂吹了哨子！”

她忽然慌了，这次轮到洛克昂出意外了吗？不是吧！吸血鬼能力那么强也会应付不来吗？

提耶利亚从袍子里抽出扫把，将它变回正常大小，骑上去在天上转了一个圆，根据哨子震动的幅度变化确定了他所在的位置——在镇护卫队的方向。难道审讯出了意外？


\
提耶利亚用最快的速度飞了过去。远远看到护卫队的楼房没有着火，她舒了口气，放下心来。能彻底杀死吸血鬼的就是烈火，既然没有着火，那洛克昂大概还没死。

在护卫队门口降落到地面，提耶利亚手里的哨子再次强烈地震了起来。她顾不得收起扫把，箭步冲到楼上，可是还未到上到二楼，她便骤然停下脚步。


\
洛克昂正悠闲地倚着栏杆扶手，像小孩子研究新鲜玩具一样把哨子放到嘴里吹一下，再拿起来端详一番，如此反复。

其实他早就听见了提耶利亚在大门外的声音。洛克昂不紧不慢地转过头，扬起手冲着气喘吁吁的提耶利亚打了个招呼：“哟，你来得挺快。我还怕这哨子不好用呢。”

洛克昂嬉皮笑脸的样子让提耶利亚气急败坏：

“不是说了遇到危险才用哨子吗！我还以为……”

提耶利亚闭上嘴，抱着双臂交叉在胸前，赌气似的把头扭到一边。

“嗯？你以为我出事了吗？欸，没想到你这么关心同事，看来合作意识提高了不少嘛……好啦我不开玩笑了。叫你来是有正事的。刚才搜查古德曼的家，找到些奇怪的东西，我不认识，审问那家伙他也不肯说实话。你读的书多，见多识广，所以让你来看看它们是什么。”


这句称赞的话倒是没有挖苦的意味，提耶利亚气消下去不少，更重要的是此时在她心里好奇心占了上风。她跟随洛克昂进入廊道左边虚掩着门的房间。哈罗正在房内等着他们，他一见提耶利亚走进房门，即刻立正行礼。提耶利亚向他点点头，眼睛早就被房间正中的桌面上摆着几只插着白色蜡烛的烛台吸引走了。它们一定就是洛克昂说的“奇怪的东西”了。


这些烛台高不到20厘米，支撑蜡烛的部分开着几个枝桠，材质不是常见的铜和铁，外表有点像干瘪的蜥蜴皮，颜色介于灰色和土棕之间，整个造型透着说不出的诡异。

提耶利亚脱掉右手的手套，左手拿起离她最近的那个烛台。她仔细地来回抚摸烛台的表面，又倒过来检查底部。



“这些都是在古德曼家的地下室找到的，”站在一边的哈罗开始说明：“还搜出来大量刀具，甚至包括砍刀，但古德曼声称它们都是手术用刀，那把砍刀是用来截肢的。”



“他肯定在扯谎。这个镇子近十年只有两个人做过截肢手术。可那把砍刀上的卷边，啧啧，”洛克昂以夸张的幅度摇摇头：“看上去可不止砍了两个人。”



提耶利亚全然不关心哈罗和洛克昂的对话内容，一心钻研着手里奇怪的烛台。只见她的脸色越来越难看，眉间的沟壑越蹙越深。

她忽然向着洛克昂平摊开手掌：“你的小刀。”

“嗯？”

“小刀，给我！”

“哦。”



提耶利亚用刀刃在烛台的底部划了一刀，她对着切开的缺口凝视了半晌，长叹了一口气，缓缓说：

“这个烛台是邪巫术的道具。它的名字是‘盗贼的蜡烛’，又叫做‘隐身烛火’，当蜡烛被点燃时，拿着它的主人就会消失，旁人看不见他的身影，也听不见他发出的声响。有了它，不需要懂巫师的隐身咒，普通人也能做到来去无踪。”

“哈，所以才叫‘盗贼蜡烛’啊，拿着它偷东西倒挺方便的。古德曼这个老色鬼，不知道有没有用它当过采花贼。嗯，不行不行，全程端着个烛台也太累了。”

提耶利亚冷着脸，用一个白眼回敬了洛克昂开的玩笑。她接下来的话，让洛克昂的笑容瞬间冻在脸上。


\
“制作它的材料，是初生婴儿的手臂。”

# 十四



洛克昂清点了一下桌上的烛台，一共有八只。

“也就是说他至少杀了四个婴儿来做这些东西？”

提耶利亚摇摇头：“这点我不敢断言。不需要是活的婴儿，使用夭折的婴儿手臂也是可以的。只是，盗贼的蜡烛只能使用一次，熄灭后便失效了。这里的蜡烛都没有点燃过的迹象。如果古德曼曾使用过它们，那么用过的烛台应该已经被他销毁了……可是，最重要的问题是，他到底是从哪里学到此等丧心病狂的邪巫术道具的制作方法。”

“这玩意儿做起来很难？”

“对于不懂巫术的普通人来说，很难。根据邪巫术典籍上的记载，制作盗贼蜡烛的关键，一是对尺骨和桡骨施咒，二是将整个手臂木乃伊化。不过，制作这批烛台的婴儿手臂，尺骨和桡骨被抽走了，不知道是用的哪种动物的骨头做的替代品。能够做到以骨易骨、魔法仍旧生效的巫师，世上怕没有几位。”


\
提耶利亚沉默了，抿着唇再次陷入沉思。洛克昂也沉默了，他懂提耶利亚的意思，他们面对的敌人不是普通的超自然生物。敌人有多强大，他们不知道。敌人是谁，他们也不知道。

“总之，”洛克昂迅速振作起来，摩擦着手掌说，“古德曼这条线索绝对不能放掉。哈罗，我们继续审那个老色鬼去。”

洛克昂走到门边。

提耶利亚抽了口气，急急转过身，冲着洛克昂的背影喊道：“我就在这附近等你。你……要小心。”

看见盗贼的烛台后，她不想、也不敢和洛克昂分开了。

洛克昂嗯了一声，给了她一个颇具安慰效用的微笑，关上了房门。

\
整个下午提耶利亚就只在护卫队的周边来回蹓跶。她思考着邪巫术、自己中的奇怪迷药、镇长被杀、人口失踪、前任天使离奇死亡。这些事情应该是相互关联的，它们被一条看不见的线杂乱地穿在一起，可她找不到线头在哪儿，不知道如何才能厘清这一切。疑团太多了。如果说这些事情背后的黑手是隐藏在暗处的超自然生物，那么他属于哪个族群？他的目的是什么？为什么选择这个平平无奇的小镇。提耶利亚越想，内心越烦躁不安。也许这种烦躁不是思考本身带来的，更多出自她对即将到来的潜藏危险的感知。


\
审讯一直进行到傍晚。哈罗把古德曼押回了地牢，洛克昂用哨子通知提耶利亚回来。

“怎么样，有进展吗？”

洛克昂撇了撇嘴，叹口气说：“那家伙狡猾得很。他承认盗贼蜡烛是他制作的。制作方法写在羊皮上，连同替换用的骨头一起锁在一个盒子里，那是他上山采药时在地里挖到的。他在扯谎！另外，古德曼坚持说他没有杀婴，制作烛台的材料取自夭折的婴儿。他说自己是出于好奇，只做了这几个烛台，连用都没用过。真该死，我们根本找不到任何证据能入他的罪。哼，连昨晚对你做的事，那家伙都有借口。他说什么你病了，给你下迷药是为了给你治病，可是你昨晚的样子……唉，至多定他一个损害尸体的罪名。”

提耶利亚想了想，说：“昨晚我跌进陷阱后，他说给我准备了好几个‘疗程’。我猜他那里应该还有为我量身定做的剩余药剂。你们不懂药草学，搜查的时候可能遗漏了。明天我们一起去他的诊所找找。就算不能用它入古德曼的罪，说不定也是个新线索。”

“一起？之前你不是说我们分开行动效率更高吗？”

提耶利亚不回答。沉默了半晌，她说：“这个小镇太危险。从现在开始，我们一刻都不能分开。”

对于提耶利亚提出的和自己时时绑定的要求，洛克昂没有表现出一丝诧异，理所当然似的接受了。

“那现在呢？我们飞回去？还是你要先吃点东西？”

“不用了，我不饿。我想走一走。”


\
暮色渐沉。他们沿着海滩缓缓前行。接近海平面的夕阳映着海面的波纹，像无数微弱的烛火在灰蓝的背景板上奄奄一息。海风有些凉，在它的鼓动下，洛克昂的披风底部时不时会敲打在提耶利亚的斗篷上。

提耶利亚说想走一走，她真的只是在走。她的眼睛盯着鞋尖，机械般迈着步长相同的步子。洛克昂知道提耶利亚有心事。他在等她开口。


\
构成海滩的材质渐渐从粗质沙砾变成了礁石。看起来提耶利亚还是不准备回到马路上，洛克昂只能跟着她一深一浅地走在随处可见的尖石之间。

经过一块巨型圆石的时候，提耶利亚忽然停了下来。她倚靠在圆石上，双手撑在身后坐了上去。洛克昂很有默契地坐到她身边。


\
提耶利亚盯着阳光消失了的海面，片刻之后，她终于开口说话。

“洛克昂，其实，我很担心。”

“嗯。”

“……我在学院里一直是优等生。我所有的认知，对世界的认识，全都是建立在书本的理论体系基础上的……”

“但是现在，你的认知受到了冲击，你碰到了解释不了的事情，对不对？”

提耶利亚有些惊讶地看着洛克昂，像是在说“你怎么知道”。

“敌人的下一个目标是我们。昨天古德曼没有得手，他只是个傀儡，以后还会有新的危险。你想说的是不是这件事？”

提耶利亚点点头。

虽说相识的时间只有短短几天，可是洛克昂第一次从提耶利亚的眼里读出“不自信”的那个瞬间，仍觉得不可思议。

“如果找不到敌人是谁，我们可能会死吧。”洛克昂语气很轻松，像是在说“明天可能会下雨”一样。他接着说：“我已经死过一次了，没什么好怕的。”

提耶利亚瞪大眼睛，仿佛听到了天方夜谭。


\
“我本来是个人类。我不是纯种的吸血鬼，是被吸血鬼吸过血，又喝下了吸血鬼的血后，才成了现在这个样子。那是太久太久以前的事了，可是就算再过一千年，我都忘不掉。那天，我和父母、还有妹妹一起去集市。那天，是劫匪屠村的日子。整个村庄都变成了红色的。一个脸上有刺青的人骑在马上，刀从我的右肩斜着劈下来。我躺在地上，人们的血淹到了我的耳朵。我动弹不得，只能睁着眼睛，看着天空飘过的云，听着一阵阵的惨叫。我听到了爸爸、妈妈的声音，然后是艾米的声音。等所有的声音，人们的尖叫、劫匪的笑声、马的嘶鸣，全都消失后，我看到了火光。

“后来的事情我没有印象了。等我醒来，我看见了一个满嘴是血的人站在我身边，后来我知道他就是力天使。整个村子都化成了灰烬。

“力天使初拥了我，他把我变成了吸血鬼。那是个很危险的操作，大部分情况下不会成功，被初拥的对象在吸血过程中会死掉。但是有什么关系，我被他找到的时候已经是活死人了。也许是因为诅咒吧，总之我活了下来。我身上流淌的血液是全新的，名字从尼尔变成了洛克昂。这颗宝石，”洛克昂轻轻按着胸前的红宝石领巾扣，继续说：“是我和过去唯一的联系。我在被烧成灰的废墟里找到了它，原本它是我母亲的一条项链。它躲过了劫匪的掠夺、也躲过了烈火的焚烧。”

\
“所以，”洛克昂又恢复了他开玩笑的那副神态，“如果提耶利亚死了，我就把你也变成吸血鬼，能不能活过来就要看你的造化了。如果是我死了，那就当终于可以结束这个被诅咒的生命了吧。”

# 十五



夜晚海风渐强，提耶利亚的斗篷兜了风鼓了起来，拍打着身后的岩石。海水的潮气贴上她的面庞，她闻到类似眼泪的咸咸的味道。

洛克昂的话让提耶利亚沉默许久。她一直不擅长与他人交流。对于别人的痛苦应该评价些什么，说什么话才能缓解空气中的沉重，她读的任何典籍中并无任何一本有相关的记载，也没有人教过她。

\
最后还是洛克昂打破了沉默：“好了，我们飞回去吧。昨晚没怎么休息，我都有些困了。”

提耶利亚想起昨晚的事，脸倏地开始发烫。好在天已全黑，脸再怎么红也看不出来。


\
二人回到驻所，在各自房间门口互道晚安。

“今晚你好好休息，我来巡逻。那，明天见。”

洛克昂转身握住房门把手。


\
“等一下！”

洛克昂闻声，回过头来，用眼神询问提耶利亚：“还有什么事吗？”

提耶利亚的嘴唇张开又闭上，来回反复了几次。踟蹰了十余秒，她终于下定决心说：“我们最好不要分开。晚上也是。我……心里很不安，总觉得会发生什么事情。”

“哎呀呀，都活了几百年了，你还害怕一个人睡？”

洛克昂挠了挠后脑的头发，企图掩饰空气中的尴尬。他眯起眼问道：“你不怕我半夜对你不轨吗？”

“你不会的。”提耶利亚的直视洛克昂，眼神透明清澈。“我不同意的话，你是不会做的。”

这次轮到洛克昂不好意思了。提耶利亚单纯又直白的回答令他开的玩笑都附带上罪恶感。


\
洛克昂躺在提耶利亚床边一侧的地板上，双手规矩地交叉在横膈肌（他本人的）的位置，身下是从他房间取出的归驻所所有的冬季棉被，铺在地上权当褥子。他从不需要盖被子，吸血鬼不畏惧寒冷。

提耶利亚在床上和衣而卧。她定好了闹钟，下命令以后不可以单独进行夜间巡逻，仿佛队长不是洛克昂而是她自己。


\
提耶利亚睡不着。她僵硬地保持侧卧的姿势，背对着吸血鬼（注意，从生物学角度来讲，被捕猎者把背对捕猎者，是件相当危险的事，请勿模仿）。房间静得令人发慌，她听见头顶的猫头鹰闹钟每秒眨一次眼睛的声音，也听见空气穿过鼻腔，和黏膜摩擦发出的轻微气流。


\
过了很久，提耶利亚猜测洛克昂大概睡着了，于是她转身换了个方向，凝视紧闭着双眼的洛克昂，用几乎不可闻的唇语说：“昨晚你救了我……谢谢。”

洛克昂像尊仰卧的雕像，一动不动。提耶利亚将视线定在洛克昂眉骨和眼睑交界的凹陷处，看了不知多久，困意终于袭了上来。


\
她久违地做了梦。她梦见洛克昂还是人类时的样子。

和吸血鬼惨白的肤色不同，梦中的洛克昂，皮肤颜色很健康，有着皮下丰富的毛细血管带来的红润和生命力。他被家人和朋友环绕在中间。不知旁人说了句什么，他爽朗地大笑，提耶利亚才知道人类由衷笑起来是这样好看。她被洛克昂的笑声感染，嘴角不自主地微微上翘。不过，在她的梦里只有洛克昂的脸是清晰的，其他人都没有面孔——这不是什么恐怖场景，也许是因为提耶利亚这辈子没认识过几个人类，对此物种的面部构造缺乏基本的想象力。


\
欢乐的画面骤然一转，鲜血和火光充斥了梦境的虚无，逼真的尖叫声震撼着提耶利亚的耳膜。周围成排的房屋被烈火吞噬，燃烧的梁木跌在地上毕剥作响。

提耶利亚感到恐惧。此时她脑子里只有一个想法——找到洛克昂，带他逃离眼前不真实的地狱场景。她四下奔跑，可是梦境里除了遍地流淌的鲜血和势头猛烈的火苗，一个人都没有。


\
“提耶利亚……”

有人在背后呼唤她的名字。声音很微弱，但她听出那是洛克昂的声音。

她回过头，见他就站在一栋被金色火光裹挟、摇摇欲坠的房屋边上，整个身体的右半部分浸在火焰里，剩下的那只左眼仍带着温暖的笑意，望着提耶利亚。

她扑过去，企图用静熄咒语把他身上的火熄灭——她忘记了，梦境里是没有魔法这回事的。

这时，两人身边的危楼轰然倾塌，卷着热浪的巨大气流朝她的面部冲击而来……


\
“提耶利亚！提耶利亚！醒醒！”

提耶利亚瞪圆双眼，从噩梦中猛然醒来。手心里全是冷汗。

她看见洛克昂焦急的脸，一只冰冷的手掌正压着提耶利亚的面颊。

有烧焦的气味。空气很热。视线清晰后，她眼角余光瞥见从门缝透出的火光。

——洛克昂的房间，正在燃烧。


\
十分钟后，洛克昂和提耶利亚走进这间从天花板到地板都被火舌吻遍了的房间。

“你的咒语真好用，烧成这样竟然一点焦糊的味道都没有。”洛克昂微笑着赞许提耶利亚刚刚施下的静熄咒语。

提耶利亚撇撇嘴，柳眉微颦嗔怒道：“真是搞不懂你，怎么现在还能笑得出来。你差一点被烧死啊。”

“我这不是还活着吗。多亏了今晚提耶利亚非要和我一起睡。这次是你救了我的性命，我们扯平了。”

说罢，他冲提耶利亚眨眨眼。

一抹绯红爬上提耶利亚的脸颊。

真该死，他连唇语都听得见。


\
洛克昂清点着他幸存下来的个人物品：定制的高档衣无一幸免，他所钟爱的古董刀的刀刃经过剧烈的化学反应变成了四氧化三铁，摆在桌上的一把东洋木刀在刀架上被平稳地分成两段。

洛克昂拿起一截木刀，惋惜地叹口气：“这火的确厉害，五分钟就把屋子里的东西全毁掉了。”

“因为它的目标是你啊，普通的火根本伤不到你吧……啊！找到了！”

提耶利亚从床底拎出半截引管，啧了一声，说：“看来是想要炸死你。”

“炸药？”洛克昂凑到跟前。他歪着头，露出一副难以置信的神情：“奇怪啊，为什么我没听见引线点燃的声音，甚至连爆炸声我都没听见。我是听见家具燃着的声音才醒来的。”


\
检查完引管的魔法残留后，提耶利亚给出答案：“它被施了静音咒，你当然不可能听见声音。”

“凶手是个巫师？”

就算多么不愿相信巫族出了个杀害天使的败类，事到如今，提耶利亚也只得承认：“……嗯，有这种可能性。”

“话说回来，”洛克昂忽然想到了什么，伸出食指在空中激动地点着，“镇长被杀的时候，我在花园里也没有听见声音。之前我还以为是棋牌室里的人动静太大，把伊奥利亚挨刀子的声音盖过去了。你说会不会，伊奥利亚的身体，或者杀他的凶器，同样被施了相同的静音咒？”

“可能是可能……就算真的被施了咒，也没有证据，时间太久了……”

“至少能肯定凶手是同一伙人。那么，隐藏在暗处的超自然生物，就在那件凶杀案的嫌疑人中间。古德曼是个突破口，我们顺着他摸下去，早晚能找到真凶。”

提耶利亚可没洛克昂那样乐观。她背靠着烧成土灰色的墙壁，用略带沮丧的声音答道：“那家伙可以毫不费力地在天使驻地安放炸药，他连天使的结界都能破解。何况敌人在暗，我们在明，只怕明天我们就会被炸到灰都不剩。”


\
灰都不剩……吗？

洛克昂若有所思，眯起眼睛，一脸神秘。

他问提耶利亚：“你会隐身咒语是吧，能不能施展在我身上？”

“不行吧……上次我在你身上用咒语……啊！不过，如果是道具的话应该没有问题，咒语配合药草学，可以将斗篷变成隐身斗篷。穿上它说不定可以隐身，只是有时间限制……你到底要做什么？”

“没关系，只要外出的时候能用就行。提耶利亚，我们将计就计——今晚天使驻所发生了爆炸，我被炸得连灰都不剩了。等下你接着在屋里放火，把这里的一切彻底烧光。”

“你的意思是……”

“这样一来，我也在暗处了。说起来，最适宜吸血鬼居住的地方，果然还是黑暗啊。”

洛克昂咧嘴笑了，他似乎对自己的计划非常满意。


\
提耶利亚不明白洛克昂葫芦里卖的什么药。

但她觉得按照他说的去做，应该不会有错。


\
他们站在房间门口，看着房内的一片狼籍在暗火的作用下彻底变为一堆灰烬。

全部结束后，提耶利亚熄灭暗火，掏出怀表看了看时间

“现在开始炼制的话，天亮前应该可以做好斗篷。不过，你一定要记住，咒语生效的时间只有两小时，而且还有不小的副作用——穿着它的人会感觉负了千斤的重量。”

“没问题，吸血鬼最不缺的东西就是力量。”

“你也一定要记住，”洛克昂指着地上一堆灰白粉末——它们在半小时前还是个灯芯草枕头——嘱咐提耶利亚说：“洛克昂已经葬身火海，灰都不剩。他的遗体，大概就在那一堆里。”

# 十六

黎明将至，镇东边海天交界处呈现出鱼背和鱼肚交界的颜色，正是星星急着赶着回家而太阳还没上班的时间点。镇上大多数人仍在美梦和酣睡中，贯穿东西的主干道静悄悄的，街面干干净净，没有一个人影。

“驾！驾！”

一架简易四轮马车踏破了阒静的空气，在石板路上笃笃狂奔。

驱车的是个有着圆脸的大个子青年，他紧紧抿着横贯脸庞的大嘴，额上渗出细密的汗珠。他的马在主人焦急的鞭策下极不情愿地交替挪动四只蹄子的位置，籍此表达它在未得到充足休息的状态下仍要超负荷加班的抗议。青年不知是过于迟钝，还是有虐待动物的倾向，他无视这匹马的正常生理诉求，坚定有力地挥动手腕，一鞭子接一鞭子抽着畜生的后背和臀部。


\
马车在一栋二层白色建筑前停下来。青年匆匆跳下车，胡乱把缰绳在门口的木桩前绕了几圈。他快步上前，使劲敲着木质门上的铜环，一边高声喊道：“洛克昂大人，洛克昂大人！出事了！洛克昂大人！”

他敲得是那样猛烈，那样急促，一个不小心，误伤了自己的大拇指。

“哎呦。”

他捂着拇指放在嘴边哈气的时候，门“吱嘎”开了。开门的不是洛克昂，而是提耶利亚。

“哦，是哈罗啊，什么事？”

“女巫大人，洛克昂大人在吗？大事不好了！古德曼，他、他不见了！”

“你说什么！有没有搞错！你们到底是怎么看守的！这么重要的犯人怎么能让他逃了？！简直是一群饭桶！”

提耶利亚的双眼喷出的怒火让哈罗虎躯一震，连着后退三步。他嗫嚅地解释道：

“昨晚负责看守的是杰克，他说自己不知道为什么特别特别困，竟然打盹睡着了。女巫大人，这事儿很蹊跷，杰克他非常负责，守夜的时候从不睡觉……他醒来后赶紧四下巡视，却发现古德曼不见了。我收到消息的第一时间就赶了过来，也许现在追捕还来得及。洛克昂大人在哪儿？他一定有办法追到逃犯。”

“啊……”

哈罗见提耶利亚的眼圈红了，隐约泛着泪光。

“出、出什么事了？”

“洛克昂……他已经……刚才我巡逻回来，发现他已经……已经……”

提耶利亚深吸一口气，努力在脸上堆砌起一副悲恸兼愤恨的表情。她接着说：“就在我外出巡逻期间，洛克昂的房间被炸得一团糟。有人在驻所放了炸弹！所有的东西都毁了，我……我甚至分不清哪里才是他的尸体。”

“洛克昂大人，洛克昂大人……呜啊，洛克昂大人！”

突如其来的哭嚎令提耶利亚手足无措。她没想到还有这出附加戏码。她到底应该和哈罗一起哭还是安慰他？提耶利亚花了1.5秒的时间思考后，果断选择了前者。

只见她一手掩面，另一只手抱住身旁的门框放声大喊：“啊啊啊，为什么……为什么他非死不可！为什么啊！”

为了让效果更加逼真可信，她硬挤出了泉涌般的两行热泪。这对女巫来说是小菜一碟，控制泪腺和控制汗腺的咒语仅相差五个字。

“女巫大人，你别伤心了，”哈罗见提耶利亚哭得比自己还伤心，反而止住了悲伤。他吸了吸快要流到上唇的鼻涕，安慰提耶利亚说：“人死不能复生。我们一定要找到杀死吸血鬼大人的凶手，为他报仇！”

提耶利亚点头，心想这孩子振作起来还真够快的。她摘下眼镜，擦干眼角的泪：

“嗯，你说的有道理。”


\
经过这么一闹，二人似乎都把正事，也就是古德曼逃跑的事情抛到了脑后。伤心的哈罗捂着胸口说想去洛克昂的生前的房间祭拜一下，提耶利亚答应了，带他一起上楼。洛克昂穿着隐身斗篷跟在两人后面。

见到洛克昂房间惨状的一瞬间，哈罗完全抵挡不住逆流成河的悲伤，扑通跪在地上号啕大哭起来。

提耶利亚用唇语对身旁的洛克昂说：“啧，你还挺受欢迎。”

洛克昂回答：“嗯，魅力太大，我也很无奈。”


\
终于等到哈罗哭累了。他起身，擦着哭成核桃的两只圆圆的眼睛。

“让女巫大人见笑了。我和洛克昂大人相识的时间不长，但他教会了我很多做人的道理……”

说着说着，他又开始抽抽嗒嗒。

提耶利亚赶紧转移他的注意力：“哦？他都教了你什么道理啊？”

“他告诉我如何才能受女孩子欢迎，虽然还没来得及实践。”

哈罗的脸红了，他那圆脸红起来就像个小苹果。

“还有，去年我家开始种土豆，老爹缺乏经验，收成不太好。洛克昂大人在变成吸血鬼之前也种过土豆，他教给我很多土豆种植的技巧。我回家后告诉给老爹听，他听了之后如梦初醒，恍然大悟。我想明年家里的土豆一定能有个好收成。唉，可惜不能继续聆听洛克昂大人的教诲了，我还想问问他懂不懂种番茄呢……”

洛克昂还真有两下子，两天内解决了哈罗的终身大事（未完成）和家族的温饱问题。


\
提耶利亚轻咳一声，清了清嗓子，开口道：

“哈罗，你冷静点听我说。洛克昂已经没了，种种迹象表明杀洛克昂的凶手、劫走古德曼的人、杀伊奥利亚的凶手是同一伙人。现在最要紧的事，就是尽快把凶手找出来，为了镇子未来的安全，也为了让伊奥利亚和洛克昂死得瞑目。我已经失去了搭档，孤掌难鸣。如果可以的话，希望你能帮我的忙，我们一起来追查真凶。”

不管这番话里真实成分有几成，至少提耶利亚的语气已经足够恳切。

哈罗被感动了。“没问题，女巫大人，我以后就听您的差遣！可是，古德曼已经跑了，我们应从何处着手呢？”

“好，既然你答应了，我就不瞒你了。其实，我们巫族可以炼制一种神奇的药水，叫做吐真剂。它能让喝下它的人诚实地回答三个问题。此种药水的原料很稀有，制作起来又麻烦。我订购的原料也是昨晚才刚刚送到。炼制一人份的药水需要一天的时间。那么接下来，我们每天审问一位伊奥利亚凶杀案的嫌疑人，让他们喝下药水，然后问他们问题，我相信一定可以找出真相。你去通知米蕾娜，我需要她做笔录。不过，药水的事情是绝对机密，除了米蕾娜、你、我，千万不要泄漏给第四个人知道，切记切记。”

“哇！”哈罗发出一声由衷的惊叹，“女巫大人你太厉害了，那个药水叫什么？吐真剂是吗？你能不能把炼药的方法教给我啊？”

提耶利亚眼睛不眨一下就拒绝了他。她给出五条理由，其中最具决定性、也最伤人的一条理由，是她认为哈罗的技能点全点在“力量”上，魔法值几乎是零。

“不过，你大可不必妄自菲薄。力量和魔法是坐标轴的两个极端，这世界上不可能有人兼备两者，连超自然生物也不可以。你应该这样想，一方面压倒性的出色，比起两方面均衡的平庸要好太多了。”

哈罗欣然接受了提耶利亚的解释。

他再度悲伤地望了望洛克昂的卧室，便毅然决然转过身，出发去市政厅找米蕾娜了。


\
洛克昂脱掉身上的斗篷，把它递给提耶利亚。

“还真有点重。”他耸了耸肩，又在肩头捶了几下。

“没想到哈罗也这么喜欢你，你看他为了你都伤心成什么样子了。”提耶利亚嘴角撇向一边，不无讥讽地说道。

“嗯？你说也？除了他，还有谁？”

“……啊，米蕾娜曾经夸过你，哦！还有、还有那些青楼的姑娘们，她们不是都很喜欢你吗？”

“原来是她们啊，可能吧……你在紧张什么啊？喂喂，你再出汗我就忍不住要咬你了啊，赶紧念那个什么咒给自己凉快凉快。”


\
提耶利亚急忙背过身，跑开十步，念动咒语。等心跳平复下来、遍体清凉后，她问洛克昂：“你这个计划真的没问题吗？万一凶手跑掉怎么办？”

“他跑掉再说，那也比我们坐在这里等死强，至少我们能搞清楚敌人到底是谁。唔，偏偏古德曼逃走了，不知会不会对计划造成影响。”

“你说，会不会是凶手把他救走的？”

洛克昂点点头，又摇了摇头：“我也猜这是凶手做的，但是，是不是救走，就不好说了。如果我估计的没错，古德曼现在已经是一具尸体了。”

# 十七



拉塞、比利·片桐、笛卡鲁特·谢尔曼、亚历杭德罗·科纳、阿里·阿尔·萨谢斯、利冯兹·阿尔马克。

# 十八

洛克昂犹豫了片刻，叹了口气，嘴角浮上一抹嘲弄的笑容：“好吧。要是真的碰到危险，提耶利亚可要记得保护我啊。”


\
临近午夜，提耶利亚骑着扫把在雾蒙蒙的天空中巡逻。她透过雾气，俯瞰小镇上点点暗红与昏黄的光，在镇子东边，一天的生活已落幕，而镇西边的夜生活才刚刚开始。

绕行一周后，她悄然降落在东南小型水坝的一角，脚尖刚刚沾地，便消失了踪影。

随后降落的洛克昂左顾右盼，压低声量喊道：“喂，提耶利亚！你在哪儿？我看不见你了。”

他感到右手的手套被扯下，一只发烫的柔软手掌紧紧捏住他的手心。

提耶利亚又出现在他眼前。

“现在能看到了吧？隐身咒只有身体接触才能破解。我需要右手执法杖，你只用左手可以吗？”

“哦……可以。”洛克昂感到自己的心脏搏动正在加快，这对于冷血生物来说是异常生理现象。

“走吧。入口应该就在附近。”

\
他们拨开藤草荆棘缠绕的石柱，找到了米蕾娜描述的“画着圆圈和宝剑”的砖石封印口，那是上一任天使施下的“安息符咒”，有防止大地震动的效果。

“咒印破坏也没关系吗？”

“没关系，其实咒语早就过期了。等下我会重新施咒。”

洛克昂点点头，伸出左手，准备破坏封印口上方的砖石，炸出一道足以让他们跻身进去的入口。他的手指才碰到画着符咒的位置，还未施力，咒印中心的砖石便碎成了一堆粉末。

出乎意料的进展让两人发出一声惊呼。

“原来这个入口已经被破坏了。”提耶利亚说。她向前探身，借着雾气笼罩的昏暗月光仔细查看剩余的咒印。

“墨迹是新的，石灰粉还有些湿润！这里是不久前才建的！”

洛克昂和提耶利亚对视了一眼，然后转过头，看着黑黝黝的洞口。

“你猜里面有什么？”洛克昂问。

“我猜……镇上丢失的那些人，应该就在里面。”提耶利亚答道。

洛克昂察觉到提耶利亚握着他的手在微微颤抖。

“嗯，我也这么想。你怕吗？要不我们别进去了，明天你让哈罗他们来查看吧。”

“当然要进去！”提耶利亚大声说，仿佛音量能让胆量同步增长。“入口封上了，说明敌人不在里面。不能等到明天，明天说不定证据就消失了。我、我才不怕呢！”


\
他们钻进洞口，沿着仄逼的台阶向下走。洛克昂能在漆黑的环境中如履平地，提耶利亚却什么都看不见。她在头顶点亮微弱的法术光，靠它的光亮勉强看清了脚下的路，心里还是没底，拉着洛克昂的手攥得更紧了。

洛克昂回头看着提耶利亚被微弱萤火照亮的脸庞。只见她眉头微蹙，聚精会神看着地下的阶梯，紧张兮兮的样子让他忽然想起昨天晚上，心头不由得一荡。



“喂，提耶利亚。”

洛克昂清了清嗓子，故作平静地发问：“你今早说我们是炮友。那作为你的炮友，我能享受什么权利，需要尽什么义务啊？”

提耶利亚恼怒地瞪了洛克昂一眼，这时候他还提什么炮友。

“我怎么知道，”提耶利亚没好气地说，“都是米蕾娜告诉我的。大概没有什么义务吧，我不是你的伴侣，不需要忍受你的刁难和脾气，高兴就在一起，不高兴就分开。如果任何一方找到人生伴侣，关系就结束了，就这么简单。”

“哦。听起来不错嘛。只要我同意，提耶利亚也同意，我们就可以发生性关系，是这个意思吗？”

“……啊，嗯，大概吧。”

“如果我有在意的女孩，那么我追求她的过程中，也就是她成为我的人生伴侣之前，我也可以和提耶利亚保持这种关系吗？”

提耶利亚从没考虑过这个问题，她的心脏忽然紧缩，血液冲上大脑。



“嗯？”洛克昂追问。

“大概……可以吧。”提耶利亚头顶的火光抖了抖。

“呵呵，那真是太好了。”


究竟哪里好了！提耶利亚忍住大声咒骂的冲动，强作镇静问道：“你有在意的女孩吗？”

“有啊。几天前才认识的，很有趣的女孩。”洛克昂的回答几乎是脱口而出。

提耶利亚的心脏再次紧缩。

“她是谁？是镇上的人吗？”

“嗯……不过她很特别，不一定会接受我啦。提耶利亚愿意和我做炮友真是太好了。”

洛克昂回过头，冲提耶利亚挤了挤眼。

“这样，在她接受我之前，有提耶利亚和我在一起，也不会太寂寞咯。”

“哼，你怎么知道我不会比你更快找到另一半。”提耶利亚嘴上还在逞强，心里开始默念控制泪腺的咒语，头顶的火光因为魔力的分散晃得更厉害了。

“嗯，说的也是。”洛克昂冲她笑了笑，露出两侧的尖牙，“炮友这种关系总有结束的一天，谁来结束都一样。”

# 二十二


提耶利亚赌气不再接话，心里琢磨着能让洛克昂在意的女孩子到底是谁。应该不是米蕾娜，这几日洛克昂没有对她有特别的表现，他暗中观察哈罗的次数都比看米蕾娜的次数多。是卖咖啡豆的娜缇，还是蔬菜店的布鲁克莉？哦，也可能是面包店的布莱德，或者是卖花姑娘露丝。仔细一盘算，这小镇上的漂亮姑娘还真不少。提耶利亚懊恼地叹了口气，她想象不出她们中间任何一个可能有拒绝洛克昂的理由。

在她胡思乱想的当间，两人已经穿过阴冷黑暗的长长隧道，来到一片开阔宽敞、由石柱支撑的地下蓄水空间。出于地形和地质的考虑，整个地下排水系统由隧道和蓄水空间交替构成，越到内部结构越复杂。相比之前蜿蜒曲折的隧道，蓄水空间的地面平坦了许多。提耶利亚凭借头顶的亮光朝四下打量了一番：高度至少有五米，左右宽约二十米，纵深未知。在如此空旷的空间里行走，甚至隐约能听见脚步的回声。


\
穿过蓄水空间，接下来又是一个隧道。这次的隧道的长度比第一个短，道路却更泥泞。穿过隧道，他们抵达了第二个蓄水空间。

两人止住脚步，他们看见自己的倒影清晰映在平如镜面的水中。这里真的有水，水清见底，深度足以没过腰部。

“怎么办，你能飞吗？”洛克昂问。

提耶利亚先点点头，又为难地摇了摇头。

“飞行的时候没法点亮法术光，我看不见……”

“唉，你们巫师还真麻烦，简直是单细胞生物。”洛克昂嘲笑道，“既然这里黑漆漆的，这件衣服也没必要了吧。”

他脱掉隐身斗篷，塞到提耶利亚怀里：“你拿着。”


“唉？啊！”提耶利亚惊呼。她脚底一空，整个人飞了起来，一手抓着斗篷和法杖，另一只手臂牢牢环住洛克昂的脖子。提耶利亚没想到吸血鬼的飞起来竟然这么快，就算横抱着一个成年超自然生物，速度也比巫师的飞行扫把快太多了。只花了三秒钟，他们已穿过水面，到达了干燥的隧道入口。三秒钟，还不够提耶利亚掏出扫把骑上去。

开始得快，结束得也突然。双脚着地后，提耶利亚仍一脸呆滞。


\
“放开吧，这种姿势没办法走路。”洛克昂低声说，嘴唇几乎贴上提耶利亚的脸。

提耶利亚窘迫地把胳膊从洛克昂脖子上卸下，头顶的法术光隐约照着她如酡的面颊。

被抱起飞行的时候，提耶利亚脑内一片空白，施下的隐身咒已经解除，此时无需接触，洛克昂也能够看见提耶利亚。

可是他还是像刚才一样牵起她的左手，说：“走吧。”

提耶利亚只低低回答了一个“哦”字，没有挣脱洛克昂的手，顺从地跟在后面。


\
第三个隧道不单止泥泞，还时不时会出现突然的下陷或绊脚的石块，两侧本应平整的石壁也像是曾遭遇过撞击般坑坑洼洼凹凸不平。提耶利亚觉得墙壁的状态很反常，上面的裂缝不是由于老化形成的自然磨损，更像地震过后留下的残迹，几处开口外缘的腐蚀程度不一，有些看上去很老旧，有些还很新鲜。她之前翻遍了小镇近十年的气象史，别说大地震了，连台风都未曾光临过这里。

提耶利亚的注意力全在这奇怪的墙壁上，没有看到挡在路中间的碎石，脚下忽然踩不稳，一个踉跄撞到洛克昂身上。

“小心。”洛克昂拉着提耶利亚的手紧了紧，扶她站稳。

“你还真喜欢往我身上撞呢。第一次见面的时候也是。”

“那次是因为地板……”

“嗯，这次是因为石头，都是外界的原因，不是提耶利亚自己撞上来的，我知道。”

洛克昂的语气透着说不出的寂寞和冷清。是地下阴冷的环境造成的错觉吧，提耶利亚想。像他这样受欢迎的人，又怎么会和“寂寞”沾边呢。


\
提耶利亚从刚才就觉得胸口闷闷的，听了洛克昂的话只觉得更加难受，嗓子像被一只无形的手捏住，呼吸不畅，焦虑不安。

“喂……”她想问洛克昂是不是也一样胸闷烦躁，没问出口的问题被眼前突然出现的场景打断了。尖利的叫喊帮助她扩张了肺泡，烦躁的气闷感觉霎时消失，呼吸通道瞬间畅通无阻。

至于提耶利亚为什么尖叫？——大概所有涉世未深的小女孩见到这样的场景，都是要尖叫的。

洛克昂倒没有出声，只猛地倒吸一大口气。


\
他们面对的是地下迷宫的第三个蓄水池，它和前两个蓄水池都不一样。这里既没有水，也不干涸。

这里有血河。

宽阔的地面上被凿出数条巨型凹槽，组成圆形的类似符咒的形状。凹槽内流淌着咕嘟咕嘟冒着泡的红色液体，空气里还飘着一股腥臭的味道。

圆形符咒的四周摆放着十二具干尸，均已干瘪到看不清生前面孔。从头发和身高看上去，应该是六具女尸，六具童尸，交叉摆放，形成环状。尸体的姿势不尽相同，有的仰面朝天双手合十，有的脸朝地面四肢蜷曲，还有骨头被扭断，做出一个活人不可能完成的姿势。他们的头全部向着圆心，像是在完成一种祭祀仪式。

祭坛的中心位置不是别人，正是之前神奇失踪的亚瑟·古德曼。几天前还在吹嘘自己对小镇有无量功德的医生，正被一众干尸围在圆心，接受顶礼膜拜。他赤身裸体盘腿僵坐，双眼浑圆，嘴巴大张，圆鼓鼓的肚子依旧腆在外面，里面本该有的眼珠、舌头还有内脏，统统都不见了。一道道交叉纵横的凹槽从他身下穿过，暗红的血水像炖煮五花肉的汤汁一样来回滚沸。

\
除了中心的十三具尸体，祭坛外围还堆着几堆白骨，不知它们曾属于人还是野兽。每一堆白骨都被一团蓝色的火焰包围，像极了荒野腐尸发出的尸火。但它们不是尸火，它们比普通的尸火更大、更蓝、更亮，像活物一样有节奏地跳动，在漆黑的殿堂里呼吸般此起彼伏，点亮了“祭坛”，为它罩上一层朦胧且惨淡的光。

提耶利亚瑟缩在洛克昂的披风后面，紧紧抱着他的胳膊，半张脸埋在他的肩上簌簌发着抖。她知道自己看上去很掉价，丢尽了巫族的脸面。

罢了罢了，现在她可不在乎什么“身价”和“脸面”，不让自己呕出晚饭已经花掉八成的力气。


\
若是没有那流动的血水和忽闪的尸火，这场景只是个静态画面，让人害怕也只是第一眼的事儿。多亏它们，为这份恐怖添上了几分灵动，让十三具尸体栩栩如生，给人绵延不断的持久惊吓。

“怕了？要不我自己下去看看？”洛克昂侧着脑袋，问身后的提耶利亚。

“你、你难道不怕吗！”

“我是吸血鬼啊，本质上就是个死人，是鬼。鬼还有什么可害怕的。”洛克昂叹了口气，吹起提耶利亚散在他肩上的额前刘海，“你放心，这里的血看起来很危险，我是不会喝的。”

提耶利亚抱着洛克昂胳膊的双臂收得更紧。“不行！我和你一起下去。下面说不定会有危险！”

\
奇怪的是，走进祭坛后，提耶利亚反倒不怕了。

她开始认真打量祭坛的每一处细节。巫师学院的图书馆有丰富的黑魔法藏书，提耶利亚读过不少。只是学院怕学生们少不更事误入歧途，将此类法术“实践”相关的部分删了个干净，只留下枯燥的魔法理论。学生们就算对黑魔法有好奇心，也顶多翻个几页，保准哈欠连天，将书扔到一边。提耶利亚自认涉猎广泛，对黑魔法的基本原理和危害有相当深入的理解。可对着眼前这诡异的尸阵，就算她背熟了魔法分类学的所有条条框框，也全然派不上用场。如同一个人背全了所有猫科动物的名称、食性、生活习性和栖息地，却从来没见过一只真猫。


\
她凝神扶了扶眼镜，从斗篷里掏出羊皮纸小本本，准备把尸阵图腾临摹下来回去慢慢研究。翻开空白页，提耶利亚忽然发现自己并没有随身携带炭笔的习惯，于是咬破了右手食指，用渗出的血在纸上画了起来，还一边画一边默念着放血咒，避免伤口的血液太快凝固。对黑魔法的强烈好奇让她画得入了神，忘记了身旁还站着一位对女巫血液极其敏感的吸血鬼。

“你……画好了没？”

“再等等。”提耶利亚没听出洛克昂的焦急难耐，她才刚刚开始临摹十三具尸体的姿势。

洛克昂只得捂着鼻子，站到隧道入口。那里是空气流动的上风口，提耶利亚的血液气味稍淡一些。


\
他听见旁边石柱的内部有响声，很轻，像风吹过松林，松叶摩擦发出的微弱声音。

“提耶利亚……”

“嗯？”

“还没画完？”

“还差一点，什么事？”

“你最好快点，这里好像快塌了。”

提耶利亚抬起头，想问洛克昂是出于什么理由得出的这个结论。她的嘴唇还没完全张开，身后的石柱便轰然倒塌，扬起的灰尘遮了她的视线，迷住她的眼睛。


\
等再次睁开眼时，她已经置身闃黑的隧道，被洛克昂裹挟着向出口狂奔，还依稀能感到身后剧烈地震带来的冲击气流与尘土。

好在地震只发生在设有祭坛的那个蓄水池，直到他们返回地下迷宫的入口，都没有其他危险发生。


\
提耶利亚终于看到了天上的星星。此刻夜深露重，雾气倒是散了。她犹是惊魂未定，没有从刚才的慌乱和奔跑回过神来，还在大口喘着粗气。


良久，提耶利亚的心情才从连续的冲击中恢复。她想起重要的事，赶紧从怀里拿出隐身斗篷递给洛克昂。

“快穿上，被人看见就糟了。”

洛克昂接过斗篷。他忽然捉住提耶利亚擎在半空的手，把她刚才咬破的那根手指含在嘴里，咬了下去。


\
“啊，疼！”提耶利亚叫着想把手指抽出来，一抬眼，对上洛克昂喷火的双目。

吸吮过后，洛克昂放开了提耶利亚的手，抬起她垂得低低的羞红的脸。

他轻咬着提耶利亚的耳垂，说：“这是开胃菜，回去之后再上主菜。”

# 二十三



提耶利亚手腕被咬开的地方已经愈合，还留存着些许疼感。洛克昂这次喝下的“主菜”分量还真不少。还好提耶利亚不晕自己的血。

之后，那件事情的激烈程度超乎了提耶利亚的想象（和第一次差不多，只是那时她没什么印象）。咒语可以麻痹掉大部分疼痛，余下的知觉对感官的冲击亦足以让她颠鸾倒凤。中途忘形之时，那个让洛克昂在意的“人类”，恍如鬼魅一般，浮现在她的脑海。

一些奇怪的念头源源不断冒了出来：

——人类的身体大概承受不了这样猛烈的拉扯和撞击吧。嘶！疼！

——舌头像冰块一样凉……人类的话，像这样被舔一遍绝对会发烧的！

——别压了！腿、腿要断了！

——不对。正因为人类很脆弱，所以洛克昂在面对“她”的时候，大概会很温柔，不会这么野蛮……

想到这儿，提耶利亚一阵哽咽，呻吟有了点哭喊的味道。

至于洛克昂，他已经完全被狂乱的兴奋淹没，没发现提耶利亚声音和心理的细微变化。


\
完事后，洛克昂的手搭在提耶利亚的后脑，把她一头柔顺的秀发当作稀罕玩具似地把玩。

“刚才……你用力有点猛。”话一出口，提耶利亚突然觉得好羞耻。她、竟然、在和洛克昂谈论这件事！

洛克昂嘴角抽动，神情很是紧张：“欸？弄疼你了？对不起……”

“没有。你伤不了我……不过，如果是人类就不一定了。”

“人类？”

“嗯，就是……”

洛克昂看着提耶利亚欲言又止吞吞吐吐的样子，忽然懂了，扑哧笑了出来。

“哦，你说我喜欢的那个女孩啊。嗯，有道理啊。天使杀害人类可是重罪，过失杀人都要蹲五百年天牢。”洛克昂装出一副很认真的样子，“要不跟她还是算了罢，还是和提耶利亚在一起比较安全。”

提耶利亚憋红了脸：“我不是这个意思……况且，如果你真喜欢她的话，跨种族什么的，不是问题。”

“哦。嗯。是啊，跨种族算不了什么……好吧，如果这次的事件能解决，我就去告诉她我喜欢她。”

洛克昂下定决心，听上去很开心。

提耶利亚沉默了。她翻身背对着洛克昂，把被角拉到肩膀，闭紧眼睛。

第一缕阳光射进窗缝前，提耶利亚终于睡着了。

\
晌午，哈罗驾着马车，穿过一眼望不到边的澄黄色田野。

田野上，大片大片的麦穗垂着头，在麦秆上摇摇欲坠，随着夏日的微风荡起层层金色麦浪。佃农们在田间挥汗如雨，舞着锄头忙于收割。看样子，今年的收成不错，是个丰收年。

马车在一间金碧辉煌的大宅前停下。

面前是一座四层高的砖瓦楼，琉璃瓦顶。它面朝广阔平原，背靠山地丘陵，前后左右全是沃田。宅子外观配色新颖别致，来客一看到它，就知道住在里面的主人实乃大俗大雅之人。门上的扣环是纯金打造，窗框看上去是镶金的，全楼的砖块涂满了罕见的防雨金色染料，整座楼像一块金色方砖，耀武扬威地插在青灰色的大山前面。


\
“原来每天半夜闪金光的建筑就是科纳家的大宅，我还以为这里有金矿呢。”提耶利亚从鼻子里发出不屑的嗤笑。

“科纳家族可不只是镇上首富！据说他们在整个东南大陆的富豪榜都能排上名号。”米蕾娜不愧是万事通，什么都知道。“别看他家这么有钱，科纳少爷和其他纨绔子弟不一样，他从不近女色，而且不管对富人还是穷人，待人接物都十分周到，还经常做慈善。大家都说他的生活方式不像富农，更像贵族。”



米蕾娜所言非虚。五分钟后，提耶利亚结结实实体验了一把贵族生活：在金碧辉煌的科纳家的客厅，坐在头层小牛皮做成的柔软沙发里，捧着镶金边的杯具饮茶，吃着裹金箔的茶点。呵，人类的“贵族”真是极尽奢华、铺张浪费。还有，愚蠢。



适才三人刚进门的时候，提耶利亚发现亚历杭德罗正对着一盘黄金和白银铸成的国际象棋棋子锁眉沉思。现在，那盘棋就在她左手边不远处的圆桌上——那桌面由象牙铺成，边缘镶着一圈金丝。

提耶利亚瞅了一眼象棋残局，问亚历杭德罗：“科纳少爷的兴趣很高雅嘛。镇子上很流行国际象棋吗？”

亚历杭德罗彬彬有礼地笑了笑，回答说：“并没有。镇上大部分人不懂棋道。这是我和朋友之前下过的一局残棋。我输了，正在苦想到底有没有破解方法。”

“你的那位朋友是伊奥利亚吗？听说前镇长很喜欢下棋。”提耶利亚抿了口茶，不经意地问。

“啊……不、不是的。我的水准不足以和镇长做对手。”

亚历杭德罗脸上起了一丝微妙的变化。除此之外，提耶利亚也注意到了他语气中的慌乱。

她开始说明来意。亚历杭德罗一边听一边点头，眉头和眼角条件反射地抽搐，又强行被拉扯舒展开——他在遮掩内心的焦虑，可越掩盖便越显得不自然。

听罢，他温文尔雅地摊开手掌，拐弯抹角又不失礼貌地表达他对一切未知药物的不信任，并声称自己只信任古德曼医生的药物，如果他要吃药，必须有古德曼医生在场。

你的古德曼医生已经变成人肉干，埋在地下三十尺了哦。提耶利亚想起昨晚的景象，肩膀一缩，打了个冷颤。


\
一直在旁默不作声的哈罗忽然开口：“这个要求难以实现。古德曼医生忽然在看守所里失踪了。我们怀疑他的失踪和伊奥利亚的死有关联。”

“什么！古德曼医生失踪了！”亚历杭德罗大声喊，他的震惊不像是装出来的。

“嗯，我们在调查中发现，古德曼疑似与之前几宗失踪案有关。但他可能不是主谋，背后还有其他黑手。至于他自己的离奇失踪……也许是事情败露，被幕后黑手杀人灭口。”提耶利亚隐瞒了黑魔法部分，其余都是事实，这是她昨晚亲眼看到的。

亚历杭德罗的双眼忽然睁得好大，像看见了什么异常恐怖的东西。提耶利亚也慌了，顺着他的目光回头看了眼洛克昂。还好，隐身斗篷还没失效。

那么让亚历杭德罗怕的，是提耶利亚刚才说的话？

贵族就是贵族。亚历杭德罗很快收拾好自己的惊讶和慌张，摸了摸鬓边梳得整整齐齐的头发，又整理了一下脖子上挂的金丝领结。

“对不起，这个消息实在太过震惊。但是，天使大人，你还没有确切证据证明医生已经遭遇不测吧？”

“嗯，的确没有。古德曼已经失踪好几天，如果他还活着，我们也想尽快将他抓捕归案。”

“好几天？他是哪天失踪的？”

\
听了提耶利亚的回答，亚历杭德罗倒吸一口气。他低下头，双手绞在一起，腿不住地抖着。

“科纳少爷？你没事吧？”米蕾娜疑惑地问，她从没见亚历杭德罗表现得如此之不淡定。

“难道你想起了什么吗？”提耶利亚也跟着逼问。

科纳抬起头，摇了摇棕金色的辫子：“没，我什么都不知道。天使大人，你刚才说喝了吐真剂之后，有一个小时的时间处理事务吧。你看，现在正是忙季，地里的事情一天都不能撒手不管，我要安排妥当今晚和明早的买卖才能放心睡觉。”

“是的，你可以用这段时间，只是不要把吐真剂的事情告诉任何人。”


\
科纳喝下了药水。他用两只手握着瓶底喝下去的，一只手托着的时候没扶稳。

接着，他像是下了很大的决心，吃力地回答了提耶利亚的问题。

这人绝对有问题——提耶利亚离开前给了洛克昂一个眼神，洛克昂心领神会地点点头。

果然，提耶利亚才回到派出所，胸口的哨子便嗡嗡大震。

循着哨声，她找到了洛克昂，他正躲在一片矮灌木里。提耶利亚刚落脚，就被他一把拉了进去。

“马上失效了，快给我换一件斗篷。你也赶紧隐身，凶手出现了！”

# 二十四



洛克昂拉着提耶利亚绕过矮灌木，沿着一条石子小路疾行。

“你们一离开，亚历杭德罗就跑到了这间屋子里，”洛克昂指着不远处的一间白色房子，“你猜猜那是谁的房子。”

提耶利亚骑着扫把飞过来的时候就有了答案，这里离伊奥利亚的宅邸不远，住在这里的应该就是那位养子。

“利冯兹·阿尔马克。”


\
白房子后院的篱笆栅门前，洛克昂拿开了他事先夹在门缝里的石子，示意提耶利亚千万别出声。两人小心翼翼绕开在后院打盹的狼狗，溜进了阿尔马克的家。

客厅传来激烈的争吵声。主要是亚历杭德罗的声音，他听上去很是激动。


\
“古德曼是不是你劫走的？前几天你说不能来和我下棋的那晚，你、你到底去哪了！你和萨谢斯到底有什么在瞒着我？”

洛克昂和提耶利亚踮脚走进客厅，躲在古董柜和墙角形成的夹缝里，凝神屏气看着室内发生的一切：亚历杭德罗一边质问一边剧烈摇晃着利冯兹纤细的身板，美少年的淡绿色脑袋像颗在风中凌乱的刺蓟草果实一般摇摆不定。

利冯兹真是好涵养，默默忍受着身体被当成拨浪鼓般的虐待。亚历杭德罗终于摇累了，一脸绝望瘫在椅子上。利冯兹靠近他，摸着身下高大男人英俊的脸庞，用尖细轻柔的声音问道：

“难道你不相信我？”

亚历杭德罗嗄声道：“我……”

“其实我是为了救你。抓捕那个女巫的计划失败了，古德曼也被擒住，可太阳炉石的净化仪式迫在眉睫，萨谢斯向我提议用你的身体——”

“什么！那个混蛋想杀我！我早就看出他觊觎我的财产！他还觊觎你！”

亚历杭德罗已出离愤怒。他疯狂地捶着桌子，唇齿间不停冒出提耶利亚听不懂的属于人类的诅咒言语。

利冯兹俯下身，一根手指封住他一双薄唇，另一只手抚摸他的胸膛，嘘声安抚。

亚历杭德罗陷进靠背，语气软了，身子也软了。


\
利冯兹一双高贵眼睛闪着如紫水晶纯净的光芒。“我不会让任何人伤害你的。”他轻声说。

亚历杭德罗的眼睛起了雾。他紧紧拥住白衣少年，下巴在那颗淡绿色脑袋上蹭来蹭去。

“你真是我的天使。”


\
接下来发生的事不可描述。提耶利亚面红耳赤，把脸转向墙壁，被洛克昂握住的手掌变得滚烫，因为紧张渗出少许汗水。

洛克昂轻轻捏了一下提耶利亚的掌心，凑到她耳边用气音说：“你别急，等晚上我们也——”

提耶利亚一拳挥到他肚子上，后半句话被一声闷哼取代。好在沙发上翻滚的那两个人的动静更大，倒也不怕被发现。

突然，陶醉在缠绵中的亚历杭德罗从沙发跌落到地上，额上渗出豆大汗珠，剧烈痉挛，痛苦万分。

“你怎么了？”利冯兹关切地叫道。

“那个……女巫给我吃的吐真剂，呵、呵，哎哟好痛，我没敢吃、吃解药，怕、怕醒不过来……”

他捂着肚子，开始满地打滚。

提耶利亚在角落里冷漠地看着亚历杭德罗痛苦的样子：真蠢，明明吃下解药睡一觉就不会有事。吐真剂是假的，毒药和安眠药才是真的。


\
“没关系，太阳炉石已经净化了，它能解你身上的毒。你等我。”

利冯兹取出一条丝帕，慢条斯理地擦拭亚历杭德罗额上的汗珠，然后起身穿好裤子，整理一下衣服，又用手指梳了梳凌乱的短发。他冲着地上痛苦不堪的情人露出一个充满怜惜与妩媚的笑容，移步走出客厅。


\
提耶利亚忍受了整整一刻钟亚历杭德罗的惨叫，利冯兹才从不知道什么地方回到他的身边。

他手里拿着一颗石头。

一见那石头，提耶利亚便感觉胃像铅坠一样难受，浑身的血液也仿佛凝结了。

洛克昂察觉出提耶利亚的异常反应，也向利冯兹所在的方向望过去。只见他手里石头有鸡蛋大小，通体流淌着暗红色的光，晶莹剔透，煞是好看。

利冯兹拿着石头在亚历杭德罗的肚子上滚了滚，石头有一瞬间呈现如渡鸦羽毛般的漆黑，接着又变成血一样的暗红。

亚历杭德罗神奇般从杀猪的痛苦中恢复了过来。

现在轮到提耶利亚面色煞白喘不上气了，连她一向温度高于常人的手掌都变得冰冷。

“我们走吧。”洛克昂用唇语提议。

提耶利亚摇头，她明白身体的异常来自利冯兹那块石头强大又诡异的魔力。她必须忍耐，她要知道它到底是什么。


\
“真的不痛了……太神奇了，这太阳炉石。”亚历杭德罗摸着肚子，喃喃自语道。

利冯兹抚摸着手中的红色鸡蛋，嘴角浮上浅笑。“呵，这只是雕虫小技罢了。它能吸收一切白魔法，再转换成更强大的黑魔法释放出来。拥有了它，就能凌驾整个超自然生物界。”

他将这颗红鸡蛋举过头顶，对着太阳，眼睛眯成一条线。阳光穿透晶石，在利冯兹的鼻梁映出一块发亮的椭圆红色胎记。

“哎呀，不好！”利冯兹忽然叫道。他将太阳炉石牢牢攥紧放在胸前：“照你这么说，那女巫正在一个个排查嫌疑人，很快就会到我这里来。现在炉石没有容器，万一被她感应到了就糟了。”

利冯兹皱眉思忖。思考结束后，他抬起头对亚历杭德罗说：

“之前说的那艘船，萨谢斯已经准备好了。今晚涨潮时分，我就把太阳炉石送回阿劳斯山。”

亚历杭德罗拉住他的手，叫道：“我和你一起去！”

利冯兹很吃惊。

他问亚历杭德罗：“你、你舍得离开这里吗？这里是你的家啊。你回去假装睡一觉，明天嫌疑就洗清了，没人会发现是你杀了伊奥利亚。你何苦跟我一起去那不属于人类的地方……”

“哼，你不和我走，就是想和萨谢斯在一起吧！”

利冯兹眨了眨眼，长长的睫毛被阳光镀了一层金：“你吃醋了？”

“是，我吃醋了。你是我的天使，永远都是我的……”

亚历杭德罗吻住利冯兹，从刚才未完成的地方再次开始。

“好……啊、嗯……那我们……呼，永远不分开……”

\
洛克昂和提耶利亚逃了出来。洛克昂的斗篷又快失效了，而亚历杭德罗和利冯兹的情事仍在继续。



他们又躲回刚才的矮醋栗灌木里，提耶利亚当场给隐身斗篷再次施法。

“还要回去吗？”洛克昂问。

“不，不用了。事情已经很清楚了，是亚历杭德罗杀了伊奥利亚，而利冯兹是超自然生物。他们晚上就要逃跑，不能让他们得逞。我们等下去趟伊奥利亚的家，必须弄清那块石头到底是何方神圣。”

# 二十五



一条狭窄的土石山路将利冯兹的白房子和伊奥利亚的大宅连在一起，只需不到半小时的脚程。今天洛克昂穿着隐身斗篷时间太长，不想再为飞行消耗体力，于是两人决定步行去往伊奥利亚的宅邸。


“喂，提耶利亚……”

“什么事？”提耶利亚没有回头看洛克昂。她怕隔山有眼，万一有人远远看到她“自言自语”的奇怪行径就糟了。

“我才知道原来人类玩的花样这么丰富。刚才他们俩做的那些，我们以后……哎哟！”

提耶利亚把胳膊抡圆了往后方狠狠甩去，刚好砸到洛克昂想用来跟她实践适才观摩到的亚历杭德罗和利冯兹“技巧表演”的部位。

“不准再提刚才看见的事！”她愤愤说道，“那是…何等失态！”

应该早一点跑出来的，提耶利亚暗自懊恼。将近一个小时，除了呻吟和污言秽语，没有一句有用的情报从利冯兹的嘴里吐出来。万一洛克昂要她在床上说那些不知廉耻的话……一想到这儿，她便不敢继续想下去。提耶利亚羞愤难耐，下巴呲溜缩进巫师袍的领子。


\
和上次造访时不同，伊奥利亚宅邸的黑漆铜门紧紧闭着。提耶利亚抬起扣环，大门发出金属相碰的清脆声音。她很快得到了回应，开门的是管家拉塞。他见提耶利亚独自到访，表情略感意外。不过除了问好，他一句多余的话都没说。



提耶利亚丝毫不客气，一坐下便直奔主题：“拉塞，我问你，你们老爷有没有提起过关于魔法的事情，比如，他可曾拥有什么法器？或者有怪异光芒的宝石？”

拉塞的嘴唇翕动了两下又重新闭合，像听不懂问题似地摇了摇头。

“唔……嗯，毕竟是那种东西，大概他不会告诉任何身边任何人……”提耶利亚撑着下巴，手指轻敲面颊，心里盘算着该如何查证。

“有了！拉塞，你去拿一个玻璃杯，再拿一壶清水，那次宴会上用的玻璃杯就可以。”

拉塞搞不懂这位天使奇怪要求背后的用意。他挠挠后脑勺，还是什么都不问，转身进了厨房，不久便将提耶利亚需要的东西端了出来。

提耶利亚拿起水壶倒了杯水，一饮而尽。她要施展的咒语可以检测出魔法的残留。可是时隔这么久，不一定有效。如果在案发时检测就好了，但那时她没想到镇上会有第二个懂魔法的生物。不过那块石头，倘若真如利冯兹所说，魔力强大到足以颠覆整个超自然生物界，说不定，说不定它会留下什么蛛丝马迹。

提耶利亚又往空杯子里倒上第二杯水，这次她没有喝，而是凑近杯壁轻轻念了几个词。而后她把杯子举起，透过这杯水观察屋内的一切陈设、地板还有天花板。

“带我去伊奥利亚的卧室。”提耶利亚命令道。


\
一手小心翼翼举着水杯，一手提着宽大巫师袍的下摆，提耶利亚一步一停走上三楼。她先透过玻璃杯看了看门锁，没有反应。已经迟了，就算案发时锁孔被施了咒，过去这么多天，也不会再有什么痕迹残留了。进入屋内，她发现地板上伊奥利亚的血迹已被洗净，书桌还保持着原有的摆设，各种文件整整齐齐码在桌前。提耶利亚仔细检查了书桌内每个抽屉，一切正常。

她把屋里每个角落都透过杯子看了一遍。

还是没有动静。

她走到窗边，让偏西的太阳光穿过玻璃杯，斜射入整间卧房。她的目光跟随阳光的路线。

有那么一瞬间，杯子底出现了米粒大小的白光。



顺着那白光的方向望去，提耶利亚锁定了伊奥利亚床边的地板。

她走过去逐一踩踏，其中一块的声音好像不太对劲。提耶利亚侧过身，向隐身的洛克昂确认。他点点头。


\
拉塞拿来工具掀开那片地板，其下方有个窄小的暗格，只有手掌宽，一个小巧精致的黑色梨木匣子藏于其中。提耶利亚掀开雕花盒盖，除了垫底的一层红色绒布，内里空空如也。

“拉塞，这里原来放着什么，你有印象吗？”

拉塞摇摇头：“老爷的床铺不需要我打理，我从不知道这暗格的存在。”

这里大概就是“太阳炉石”之前的藏身地点吧。提耶利亚已有十成把握是利冯兹差遣亚历杭德罗杀了伊奥利亚，杀人动机不是别的，就是那流淌着怪异魔力的石头。

可是那石头的来历、作用，为什么会收藏在伊奥利亚的卧房，她还是一无所知。

只剩下这空荡荡的匣子。

提耶利亚叹了口气，收好匣子，与拉塞告别。

“天使大人，杀害老爷的凶手找到了吗？”走到门口时，拉塞让人意外地开声询问提耶利亚。

提耶利亚略微犹豫，还是摇摇头，给了他一个否定的答案：“还没有。不过你放心，我一定会将凶手绳之以法。”


\
傍晚，他们回到驻所。提耶利亚把黑梨木匣子摆在面前，支棱着脑袋对着它发呆。

按照利冯兹说的，太阳炉石能吸收一切白魔法……

（怎么会有这种东西存在于世上，怎么可能……）



“怎么，提耶利亚也有想不明白的事？”

提耶利亚转头，见洛克昂倚靠在她桌子的边缘，侧身低头，冲她微笑。她脸一红，嗔道：“我又不是什么都懂！”

她接着说：“今晚我怕……我的法术帮不上什么忙……”

“一切交给我好了。他们想要出海，到时候萨谢斯也会在那艘船上。提耶利亚只需要在明处，假装施法掩护我，我来控制重力和海浪打翻他们的船。只要拿到那颗石头做证物，就能证明是他们杀了伊奥利亚，还有那些被献祭的人类。”

计划听起来不错，提耶利亚对洛克昂的能力毫不怀疑，但仍隐约觉得不安，怕有无法计算到的变数。


\
“等抓住了那帮混蛋，我就去和之前说的那个女孩子表白，你说好不好？”洛克昂挺直身子，面朝门口轻声说道。

“这种事别问我，”提耶利亚没好气地回答，“况且，在战斗之前说这种话，通常会死得很惨。”

“不问你问谁？现在除了你，别人都认为我死了，就算我想问哈罗也不可以啊。”洛克昂说着，从怀里掏出一个白色的圆形吊坠，放在桌上，推到提耶利亚的眼前，“这是我以前向她要的肖像，你帮我出出主意。”

吊坠白色珐琅质，外围镶一圈金丝，中心有精致的粉色蔷薇雕刻，只有方寸大小，古色古香。看上去不像是市面上常见流通的肖像坠子，搞不好还是个古董。能用得起这种货色，原来洛克昂喜欢的是个大户家的小姐。

提耶利亚特想把坠子甩到洛克昂的脸上，再说几句挖苦讽刺的话，可该死的好奇心还是令她屈服了。她悻悻地拿起吊坠，不服气地想着“我倒要看看她能有多好看”，撅着嘴把盖子打开。


\
提耶利亚的眉毛瞬间拧到一块儿。她不解地抬头看着洛克昂，双唇微启，好像想要问什么。

洛克昂仍旧微笑着直视她的脸。

提耶利亚明白过来。她合上吊坠，头垂得不能再低。


\
“怎么样？提耶利亚有什么意见吗？”

“我……我不知道。”

“是吗？好可惜，连提耶利亚都不知道的事情，那世界上没有其他人会知道了。”

“……”

“你觉得她会答应吗？”

“……你、你到底喜欢她什么啊？”提耶利亚盯着书桌抽屉小声发问。

“她让我感觉自己还活着。”

提耶利亚忽然忘了脸红，抬起头怔怔望着洛克昂。

洛克昂取回她手中的吊坠。“明天再说吧。也许到时候你就有答案了。”

# 二十六



潮水从后半夜开始上涨。蓝黑的夜空里月色暗淡，亦无繁星耀眼。提耶利亚在港口附近教堂倾斜的屋顶隐身坐稳，静待洛克昂传来寻到利冯兹一行人踪迹的信号。

\
提耶利亚看看手里的哨子，叹了口气，下巴抵住膝头。关键时刻自己偏偏像个废物，既不能隐身飞行探测敌踪，攻击魔法也可能起不了丝毫作用。从刚刚上任时想要争取领导权，到现在什么事情都习惯依靠洛克昂，提耶利亚想破脑袋也想不通到底哪件糗事是转折点。只是，她竟没有一丝居于人下的不甘心，反而觉得有洛克昂在真是太好了。这么短短几天，她以往的争强好胜都跑到哪里去了？

然而比起好胜心的丢失，更糟糕的是她魔法信仰的崩塌。提耶利亚一直瞧不起正统魔法之外的法术，还有那些游离于学院派之外的乡野巫师。伟大的巫师，其姓名必定会刻在学院的碑塔之上，这也是她曾经的目标。白天的太阳炉石，还有看见它后身体擅自做出的不适反应，甚至利冯兹身上不可思议的强大魔力……一股未名恐惧倏然涌入心脏，提耶利亚不经意打了个哆嗦。她隐约有种感觉。那块被称作太阳炉石的石头，也许是魔法的起源，是巫师这种生物存在的理由，它可以操纵接近者的心，甚至吞噬他们的灵魂。



灰蓝的月亮西斜，挂上西山顶峰的树梢。恍惚不安中，提耶利亚望着远方海天一色处的水平线。今晚风轻浪平，风向东偏北，若是施过法术的船，一旦驶离港口，必定如拉紧弓弦射出的箭。不知洛克昂操控重力的有效范围究竟有多远，能不能有足够的时间让造出的巨浪掀翻船体。要是她能帮上什么忙就好了。

半残的月亮降到了树桠，手中的哨子终于震起来。

\
赶到西边岬角时，提耶利亚没有见到利冯兹一干人的影子。洛克昂朝悬崖边的黑水努了努嘴，示意他们就在崖底洞穴内。提耶利亚悄然飞起，踏上崖壁上一块较为平坦的岩石，站稳后朝下望去。她打定主意，等利冯兹一出来就对船只施展巫火。管它有没有用呢，就算不能将船舷击穿一个大洞，至少也能让他们措手不及，给洛克昂偷袭的机会。

提耶利亚专心凝视着缓缓击打峭壁的黑水。突然，崖边冒出一阵沉响，提耶利亚登时紧张起来。一条长影滑出来，浮于海面之上。就是它了！提耶利亚念动咒语，火光由法杖顶端跃出，撕破漆黑的空气，直向船尾射去。


这一击注入了极限的爆发魔力。没有持久战的必要，一旦利冯兹使用太阳炉石，提耶利亚能不能施展出魔法都是未知，不如孤注一掷。耀眼的火球像浸过桐油熊熊燃烧的箭矢向着悬崖下方疾驰，眼看就要追上船身，却突然如捻熄的蜡烛般归于黑暗。


\
“还是走漏了风声吗？你竟然追了过来。”利冯兹阴冷且轻细的声音和海浪声一起传进提耶利亚的耳朵，“你叫提耶利亚，是吗？我劝你别费力气了，我没有杀你的理由，别让我觉得你是个麻烦。”

话里的意思很明显，对利冯兹来说提耶利亚就是个战五渣，连出手的必要都没有。

这个混蛋！从来没有被人这样轻视过的提耶利亚怒火中烧，嘿嘿冷笑一声，怒斥道：“没有杀我的理由？几天前派那个猪头设下陷阱抓我的就是你吧！”

“呦？这么说你知道了。其实没别的意思，我和你无冤无仇，只是觉得你的血更纯净一点罢了，不过也不是非你不可。趁着现在我不想杀你，我劝你见好就收吧。可以继续回去做你的天使，接受人类的崇拜。”

说话间，夜晚的薄雾慢慢散去，提耶利亚察觉到利冯兹的船有些不对劲。明明是涨潮，远处的海面还在浪打浪，但那船体周围的水却似冰封一般不为所动。


\
不对！他不是不想杀我，是不能杀我！他的所有魔力都用来为那艘船编织结界，想要冲我发起攻击，结界便会消失，船也要跟着出事。原来敌人也自顾不暇，提耶利亚忽然升起信心。



“太天真了，你以为区区结界便能挡住我吗？”提耶利亚用挑衅与煽动语调嗤笑利冯兹，“我能够穿透结界，你信不信？

她朝洛克昂所在的方向瞄了一眼。洛克昂站在海水里，浪花的泡沫线来回卷着他的裤腿。他迅速点头，示意提耶利亚一切已准备就绪。


提耶利亚张开双臂，法杖夸张地在空中划了一个圆，用足以让利冯兹听到的音量朗声念出巫水咒语。她故意在咒语中额外加了一些气音，好让它听起来更加怪异。



霎时，海面上掀起两米高的巨浪，向利冯兹所在的船身扑过去。时机刚刚好，提耶利亚暗笑，再次念出经由她二次创作改编的巫水咒。



巨浪一道接着一道，使近海海域形成几眼漩涡。水的翻搅煽动空气，在海岸线野蛮生长的海芦苇被气流压得趴倒一片。疾风鼓起双臂托举的提耶利亚身后的斗篷，有一瞬间，她产生了一种错觉——以为这水、这风真是因她而起。



利冯兹并不好对付。他的船被海浪卷到半空又跌下，从一个漩涡掉进另一个漩涡，但结界始终完整，船体没有一丝毁坏。船上的三名乘客也无一人被甩出船身之外，就像被封印在水晶球里的玩偶一样安全。至少有两个是安全的——亚历杭德罗·科纳只是个人类，肉身哪经得住如此剧烈的折腾，早已晕厥过去。



洛克昂的眼睛死死盯在阿里·阿尔·萨谢斯身上。意识不清的科纳在圆形的结界中像弹球滚来滚去，而萨谢斯只是从容地抓住船舷，胡须上的嘴角微翘，露出轻蔑的笑。这怎么可能！除非……

这时，萨谢斯开口说话了。

洛克昂凝神，一边继续操控海水的重力，一边摒弃周围滔天的水浪声和万马奔腾的风声，将敏锐的听力集中到船体的方向。

萨谢斯对利冯兹说：“这样下去谁都走不了啊。你想想办法。”

“再忍忍。我不信她的魔力还能持续多久，她是个劣质品。”

“劣质品？不见得吧。我没见你们中间哪一个能把海水操纵到这种程度。大帅，我看你是舍不得这船上的蠢货人类吧？结界你还能维持多久？万一天亮我们还走不了，一切就完了。你想和人类一起完蛋，可别拖我下水。不如这样，你把炉石给我，由我去带回给那位大人……”

“别开玩笑了！”洛克昂听到利冯兹粗暴地打断萨谢斯，“我知道了，你准备好——”



准备好什么？洛克昂没听见，也可能利冯兹压根没说。电光火石间，颠簸在巨浪间的船体崩裂成无数碎片，科纳被被甩出数十尺，萨谢斯不见了踪影，而利冯兹正由半空飞身下坠，并向提耶利亚所在方向伸出手掌。



由他掌中射出一道烈焰，一径穿透了扑面而来的浪头，空气登时弥漫着咸湿的水蒸汽。如果将刚才的巫火比做箭矢，那此时的便是火枪——和它相比，刚才提耶利亚使出全力释放的巫火不值一哂。



提耶利亚来不及躲避。躲也是徒劳，这是巫火。除非有抵挡或熄灭巫火的能力，否则它会跟你到天涯海角，直到将你焚到渣都不剩一粒。



面对这辈子未曾见识过的强大“乡野”魔法，提耶利亚做了明智的选择，也是最简单的解决方案：闭上眼睛，等死。

哦，还没给洛克昂答案。

唉，算了吧。


\
提耶利亚感到四周嘈杂的声音忽然停了，像交响乐末尾干净利落的终止符。

还是能听到一丝水声……有海的味道……是死后的灵魂跌入海中了吗？

等等！血腥味？！

提耶利亚猛地惊醒。她双眼睁得浑圆，怔怔望着挡在面前的那个人。

骗人！


\
洛克昂低头看了看胸前焦黑的空洞，一丝苦笑爬上嘴角。

“明天、果然还是太晚了……”

他终于支撑不住，直挺挺跌下悬崖。

# 二十七



提耶利亚跟着跳了下去。

潮水仍在上涨，她没花费太多时间就找到了半死不活的洛克昂。不知是不是因为他的心脏已经灰飞烟灭，提耶利亚拖着他的身体时，感觉他好轻——比昨晚他压在她身上的时候轻多了。

又或许，消失的是他灵魂的重量。

提耶利亚强迫自己不要乱想，族裔学教授说过吸血鬼没那么容易死。

可是她又记起教授说的，杀死吸血鬼很难，除非是——

烈火。


\
提耶利亚脱掉自己的斗篷（谢天谢地，她在战斗前给斗篷施了避水咒），让洛克昂平躺在上面。

冷静。要冷静。她深吸一口气，目光躲避开洛克昂胸前被烧成焦炭的洞。那里太可怕，洞口露出的白骨和血管都是黑色的。他能够尚存一口气，完全是沾了凉血生物代谢缓慢的光。



“洛克昂……我该怎么办？”

洛克昂的鼻息越来越弱，他的生命正随着时间一同流逝。提耶利亚握紧他垂在体侧的手掌。他手掌的温度比往常更冰冷，冷到她与之触碰的手指禁不住一阵发抖，她的指节碰到一件硬物。

那是洛克昂之前在镇上铁匠铺购买的刀子。

呵，她曾经害怕他用这把刀割开她的皮肤，喝她的血。



是了！女巫的血！



提耶利亚迅速从洛克昂的裤兜中掏出那把价值40银币的折叠刀，捏住刀柄，展开刀刃，刀锋反射出一道寒光。

她深吸一口气，举起刀，对着左手手腕内侧用力划下去。血涌了出来，速度很快。

提耶利亚用右手撬开洛克昂虚弱无力的牙齿，将伤口按了上去。

可是，她的血液没有进入他的身体，而是沿着嘴角流下来。他连吞咽的力气都没有了。

“求你了，快喝吧。”提耶利亚哭了出来。


\
（你真的要这么做？）

——脑海里响起一个声音，听起来像是提耶利亚自己的声音。

是的。她回答那个声音说。

（你要付出代价的。）

我不怕。不管什么代价，我都会付。

（你也许会后悔。）

不，我不会。

提耶利亚身下有风升起。这时，洛克昂的喉咙动了一下。

他的求生本能被激发了出来，原本紧闭的眼皮开始颤动，冰凉的舌头舔舐提耶利亚伤口内粗糙的血肉，他找准静脉血管，大口大口吸吮。

他吸吮的力度渐渐加大，并用尖牙咬开另一道伤口。提耶利亚发现那胸前被灼烧的血管和骨头开始变色，从乌黑变为正常的红色和白色。

洛克昂的身体正在提耶利亚的血液作用下自我修复，但速度太慢了，静脉血的流速不足以支撑他修复消失的心脏。


\
提耶利亚扯开衬衫衣领，将落在左侧颈肩的发梢悉数撩到右侧，让左颈处的汗腺发挥它们应有的作用。等到汗水打湿领口，她抽回被洛克昂含住的左腕，开始吻他。

洛克昂口中满满都是她血液的咸腥气味。

她在吻的间隙中低语：“来吧，洛克昂，这里……”



洛克昂忽然睁开眼，瞳孔周围如碧绿的火焰。他猛地坐起，攥住提耶利亚后脑的头发向后一拉，嘴唇贴上她裸露的光滑颈部皮肤。



提耶利亚的颈动脉处一阵刺痛，鲜红的血液在心脏收缩的作用下从出口喷薄涌出。她左手搂住洛克昂的后背，右手环在他的腰上，将他抱紧，鼓励他喝得更大口一些。

提耶利亚眼睛开始模糊，但身体的感觉很愉悦。海浪每次冲刷，变为心脏的一次跳动，心脏的每次跳动，又变为洛克昂的一次吞咽，洛克昂的每次吞咽，最终会变为他新生心脏的一部分。

她左手指尖碰到他新长出的肌肤，是记忆里的温度，凉得好像春季清晨的雾霭。原来这就是生命的交换，真好。



四周一片沉寂，只剩她胸腔心脏沉重的搏动声。

渐渐的，连心脏的搏动也消失了。




\
三天后，提耶利亚再次睁开眼。她已经躺到了自己的床上，身上是平时的睡衣。她看见天花板在转着圈圈，依稀有一群金色的小天使飞过。这让她怀疑自己已经死了。



“提耶利亚！你醒了！”

一张熟悉的脸凑了过来，赶走了满天的小天使，那像石膏一样白的脸上还有两个黑眼圈。

“你也、会有黑眼圈？”她张开口，气若游丝。提耶利亚这才发现自己连说话的力气都没有。

“三天没睡，谁都会有黑眼圈啊。浑身是毛的狼人都不例外，只是你看不出来而已。”

除了黑眼圈，洛克昂看上去一切正常。提耶利亚牵动嘴角，想尝试笑一下，可是很快便放弃了。

她的嘴唇很干，轻轻一扯都有些疼。

“别说话了，你现在很虚弱。要水吗？”洛克昂问。

提耶利亚眨眨眼。


\
洛克昂举着水杯，站在床边犹豫了一下，仰起头自己喝了一大口，然后轻捏提耶利亚的双颊，嘴对嘴将水喂了进去。提耶利亚一惊，不过以她现在这副德行是没办法反抗的，只能抵着舌尖迎接从洛克昂嘴里流下的清水。

等有力气了再收拾他吧。



用这种方式喝完整杯水，提耶利亚也差不多再次筋疲力尽了。

“你接着休息吧，我就在这里，哪儿也不去。镇子上的事情我交给哈罗了，你不用担心。”

提耶利亚眨眨眼，眼皮甚是沉重。



还有一件事。

她用尽剩余的力气，努力说出她想要告诉洛克昂的话：“那件事……我想、她……大概，会……答应的。”

说完，她再次沉沉睡去。没有遗憾了，随她睡多久吧。

阿劳斯山位于整个世界的最西端，再往西去便是无尽的海洋。

它坐落一个火山岛上，遍地是灌满熔岩的湖和金属铸成的地面。

那是魔族的圣地，不过世上的魔族只听过它的大名，鲜有人知晓它的位置，更别说亲临朝圣了。


\
阿劳斯山正东部岸边，海水经年被铁矿浸泡，呈现血的红色。一只大白鲨冒着搁浅的危险向岸上游去。眼看它的肚皮就要贴上铁石海底，白鲨忽然消失不见。一个浑身湿漉漉的白衣美少年，款步走上海岸。

他从嘴里吐出一块鸡蛋形状的红色晶石，紧紧握在手中，然后冲身后喊道：“出来吧。”

阿里·阿尔·萨谢斯出现在他身后。红发男人走到少年面前，嘴角向上扯动，说：“大帅辛苦了。多谢大帅的顺风车。”不过言语里并没有多少感激之意。


\
利冯兹看了他一眼，面无表情。

“走吧，雷杰尼已经到了。”他说。

绕过岩浆涌动的火山口，顺着山路下行，他们来到山阴处的峡谷。那里是阿劳斯山的核心处，是一处绿洲。

火山和矿脉中心的绿洲，怎么想都是不可思议的奇事。

\
一个面容和提耶利亚一模一样的白衣少年正端坐在清澈的池水旁，望着水里悠闲自在的鱼。

“你在发什么呆呢？”利冯兹走上前问道。

“我在想晚上吃哪一条。”酷似提耶利亚的少年笑着抬头，看了利冯兹一眼，又低下头看鱼。

“容器找到了吗？”

“没有，哪有那么容易。全都不堪一击。不过我已经有了下一个目标，半年后再说吧。你呢？去了这么久，这次应该拿到手了吧。”

“嗯。到手了。”利冯兹装得一脸平静。

“恭喜你啊，”另一个少年拍了拍手，“这可是大功一件，权天使大人会很高兴的。”

“对了，雷杰尼。我这趟出去，好像见到了你的双胞胎妹妹。也可能不是，不过她和你实在太像了。”

“你说什么！你为什么不把她带回来！”雷杰尼一直悠哉悠哉的脸上忽然有了怒容。

“一个劣质品，带回来有什么用。别忘了，炼金炉里吸掉所有精华的是你啊，你当初可是什么都没给她剩下。”

“……她还好吗？”

“我不知道。只要她能躲得过我的巫火，也许现在过得还不错。”

“你……”雷杰尼嘴角抽搐。

利冯兹轻叹一口气：“没办法，谁让她要和我作对，要抢太阳炉石呢。就算是权天使大人也不会同意留下她吧。啊，你瞧，大人马上就到了。”利冯兹伸手指着发红的天空，“萨谢斯，我们先去议事厅准备。”

利冯兹挑了挑柳眉，抛给雷杰尼一个属于胜利者的微笑，转身大步走入林间，他能感到同僚利剑似的目光正剜着他的后背。

雷杰尼攥紧膝上的衣角：“利冯兹·阿尔马克……”
